## إعداد GroupDocs.Signature لـ Java

إعداد GroupDocs.Signature سهل للغاية. يمكنك تضمينه في مشروعك باستخدام Maven أو Gradle، أو تنزيله مباشرةً من موقعهما الرسمي. إليك الطريقة:

### استخدام Maven
أضف التبعية التالية إلى ملفك `pom.xml` ملف:
```xml
<dependency>
    <groupId>com.groupdocs</groupId>
    <artifactId>groupdocs-signature</artifactId>
    <version>23.12</version>
</dependency>
```

### استخدام Gradle
قم بتضمين هذا في `build.gradle` ملف:
```gradle
implementation 'com.groupdocs:groupdocs-signature:23.12'
```

إذا كنت تفضل ذلك، قم بتنزيل الإصدار الأحدث من [GroupDocs.Signature لإصدارات Java](https://releases.groupdocs.com/signature/java/).

## دليل التنفيذ

### تحميل مستند من InputStream

تتيح لك هذه الميزة تحميل المستندات ديناميكيًا دون الحاجة إلى تخزينها محليًا. إليك كيفية تطبيق هذه الميزة:

#### إنشاء مجرى الإدخال
أولاً، قم بإنشاء `InputStream` لملف PDF الخاص بك:
```java
import java.io.FileInputStream;
import java.io.InputStream;

InputStream stream = new FileInputStream("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF");
```

#### تهيئة التوقيع باستخدام InputStream
بعد ذلك، قم بتهيئة `Signature` الكائن الذي يحتوي على مجرى الإدخال الذي تم إنشاؤه:
```java
import com.groupdocs.signature.Signature;

try {
    Signature signature = new Signature(stream);
} catch (Exception e) {
    throw new Exception(e.getMessage());
}
```
تتيح لك هذه العملية العمل بشكل مباشر مع تدفقات المستندات، مما يوفر المرونة في كيفية الوصول إلى المستندات ومعالجتها.

### توقيع مستند باستخدام خيارات رمز الاستجابة السريعة

بعد تحميل المستند، لنوقّعه باستخدام خيارات رمز الاستجابة السريعة (QR code). تُحسّن هذه الطريقة أمانك بتضمين بيانات إضافية في توقيعاتك.

#### إنشاء كائن التوقيع
تهيئة `Signature` غرض التوقيع:
```java
Signature signature = new Signature("YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF");
```

#### تحديد خيارات علامة رمز الاستجابة السريعة
قم بإنشاء وتكوين خيارات علامة رمز الاستجابة السريعة لتحديد البيانات التي ترغب في ترميزها في رمز الاستجابة السريعة:
```java
import com.groupdocs.signature.options.sign.QrCodeSignOptions;
import com.groupdocs.signature.domain.qrcodes.QrCodeTypes;

QrCodeSignOptions options = new QrCodeSignOptions("JohnSmith");
options.setEncodeType(QrCodeTypes.QR);
```

#### تعيين الموضع وتوقيع المستند
حدد المكان الذي يجب أن يظهر فيه رمز الاستجابة السريعة (QR) على المستند، ثم قم بتوقيعه:
```java
options.setLeft(100);
options.setTop(100);

String outputFilePath = "YOUR_OUTPUT_DIRECTORY/signedSample.pdf";
signature.sign(outputFilePath, options);
```
This step embeds a QR code containing \"JohnSmith\" at coordinates (100, 100) on the document.
