## دليل التنفيذ
### توقيع مستند PDF باستخدام ملصق نصي
يوضح هذا القسم كيفية إضافة توقيع نصي إلى مستند PDF باستخدام GroupDocs.Signature لـ .NET. تتضمن العملية تحديد خيارات التوقيع وتكوين إعدادات المظهر.

#### الخطوة 1: إعداد مسارات الملفات
قم بتحديد مسارات الإدخال والإخراج لملفات PDF الخاصة بك:
```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF";
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/PdfSticker.pdf";
```

#### الخطوة 2: تهيئة كائن التوقيع
إنشاء `Signature` الكائن الذي يحتوي على المسار إلى مستند الإدخال الخاص بك.
```csharp
using (Signature signature = new Signature(filePath))
{
    // سيتم وضع رمز التوقيع هنا
}
```

#### الخطوة 3: تكوين خيارات علامة النص
قم بتحديد وتكوين خيارات علامة النص لملصقك:
```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    Left = 50,
    Top = 200,
    Width = 100,
    Height = 30,
    
    SignatureImplementation = TextSignatureImplementation.Sticker,
    
    Appearance = new PdfTextStickerAppearance()
    {
        Icon = PdfTextStickerIcon.Star,
        Opened = false,
        Contents = "Sample",
        Subject = "Sample subject",
        Title = "Sample Title"
    },
    
    Margin = new Padding() { Bottom = 20, Right = 20 },
    ForeColor = Color.Red,
    Font = new SignatureFont { Size = 12, FamilyName = "Comic Sans MS" }
};
```
**توضيح**:هنا نقوم بتحديد الموضع (`Left`، `Top`)، مقاس (`Width`، `Height`)، ومظهر الملصق. `SignatureImplementation` تحدد الخاصية أن هذا هو توقيع ملصق نصي.

#### الخطوة 4: تنفيذ التوقيع
اتصل بـ `Sign` طريقة تطبيق التوقيع:
```csharp
SignResult signResult = signature.Sign(outputFilePath, options);
```
ستؤدي هذه الطريقة إلى حفظ المستند الموقع في مسار الإخراج المحدد.
