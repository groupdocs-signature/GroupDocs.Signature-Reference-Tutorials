## دليل التنفيذ

دعونا نقسم التنفيذ إلى أقسام قابلة للإدارة.

### نظرة عامة على ميزة التحقق من توقيع النص

تتيح لك هذه الميزة التحقق من توقيعات النصوص في المستندات، والتأكد من مطابقتها للمعايير المحددة. يمكنك تخصيص خيارات التحقق، مثل تحديد الصفحات المطلوب فحصها ونمط النص المطلوب البحث عنه.

#### الخطوة 1: إنشاء طريقة التحقق

ابدأ بتحديد طريقة لتغليف منطق التحقق الخاص بك:

```csharp
class SignatureVerification
{
    public void VerifyTextSignature(string filePath)
    {
        using (Signature signature = new Signature(filePath))
        {
            // سيتم وضع رمز التكوين والتحقق هنا
        }
    }
}
```

#### الخطوة 2: تكوين TextVerifyOptions

حدّد خيارات التحقق من توقيعات النصوص. هنا، ستحدد الصفحات المطلوب التحقق منها ونمط النص الدقيق.

```csharp
TextVerifyOptions options = new TextVerifyOptions()
{
    AllPages = false,
    PagesSetup = new PagesSetup() { FirstPage = true },
    Text = "Text signature",
    MatchType = TextMatchType.Exact
};
```
- **جميع الصفحات:** تم ضبطه على `false` إذا كنت تريد التحقق من صفحات معينة.
- **إعداد الصفحات:** حدد أرقام الصفحات أو نطاقاتها. هنا، نتحقق من الصفحة الأولى فقط.
- **نص:** نمط النص الذي يجب أن يتطابق داخل المستند.
- **نوع المباراة:** يحدد مدى دقة مطابقة النص؛ هنا، يتم ضبطه على `Exact`.

#### الخطوة 3: إجراء التحقق

تنفيذ التحقق ومعالجة النتائج:

```csharp
VerificationResult result = signature.Verify(options);

if (result.IsValid)
{
    Console.WriteLine("\nDocument was verified successfully!");
}
else
{
    Console.WriteLine("\nDocument failed verification process.");
}
```
- **نتيجة التحقق:** يحتوي على تفاصيل حول نتيجة التحقق.

### نصائح استكشاف الأخطاء وإصلاحها

- تأكد من أن مسار الملف الخاص بك صحيح لتجنب `FileNotFoundException`.
- تأكد من أنماط النص إذا فشلت عملية التحقق بشكل غير متوقع.
- تأكد من أن لديك الأذونات المناسبة لقراءة الملفات.
