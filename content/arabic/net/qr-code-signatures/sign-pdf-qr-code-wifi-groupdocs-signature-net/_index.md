## دليل التنفيذ

### الميزة: توقيع المستندات باستخدام رمز الاستجابة السريعة

توضح هذه الميزة كيفية التوقيع رقميًا على مستند PDF باستخدام رمز الاستجابة السريعة QR الذي يحتوي على إعدادات WiFi.

#### الخطوة 1: تحضير مسار المستند وموقع الإخراج
```csharp
string filePath = @"YOUR_DOCUMENT_DIRECTORY\SamplePDF.pdf";
string outputFilePath = System.IO.Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "SignedSamplePDF.pdf");
```

#### الخطوة 2: إنشاء رمز الاستجابة السريعة (QR) بمعلومات WiFi

باستخدام `QrCodeSignOptions`، حدد إعدادات WiFi الخاصة بك:

```csharp
using GroupDocs.Signature.Options;
using GroupDocs.Signature.Domain;

// قم بتحديد بيانات اعتماد WiFi المراد تضمينها في رمز الاستجابة السريعة QR.
var wifiInfo = new QrCodeWiFi(QrCodeWiFiFrequancy.FiveHundredMhz, "YourNetworkSSID", "password");

QrCodeSignOptions options = new QrCodeSignOptions(wifiInfo)
{
    EncodeType = QrCodeTypes.QR,
    Left = 100,
    Top = 100
};
```

#### الخطوة 3: توقيع الوثيقة

استدعاء `Sign` طريقة تطبيق توقيع رمز الاستجابة السريعة QR:

```csharp
// توقيع وحفظ الوثيقة.
signature.Sign(outputFilePath, options);
```

### نصائح استكشاف الأخطاء وإصلاحها:
- تأكد من صحة مسارات الملفات الخاصة بك لتجنب `FileNotFoundException`.
- تحقق من إعدادات WiFi (SSID وكلمة المرور) للتأكد من التشفير الصحيح.
