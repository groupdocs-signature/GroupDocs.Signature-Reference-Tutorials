# كيفية توقيع مستند PDF باستخدام رمز الاستجابة السريعة للبريد الإلكتروني باستخدام GroupDocs.Signature لـ .NET

## مقدمة

في عصرنا الرقمي، أصبح ضمان صحة وسلامة المستندات أكثر أهمية من أي وقت مضى. تخيل أنك تحتاج إلى مشاركة معلومات حساسة بأمان داخل مستند لا يمكن الوصول إليه إلا من قِبل أشخاص محددين، وهنا تبرز أهمية توقيع المستندات ببيانات مشفرة. سيرشدك هذا البرنامج التعليمي إلى كيفية استخدام GroupDocs.Signature لـ .NET لتوقيع مستندات PDF باستخدام رمز الاستجابة السريعة (QR code) الذي يحتوي على عنصر بريد إلكتروني، مما يوفر لك الأمان والراحة.

**ما سوف تتعلمه:**
- كيفية إعداد البيئة الخاصة بك لاستخدام GroupDocs.Signature لـ .NET
- خطوات إنشاء وتكوين رمز الاستجابة السريعة الذي يحتوي على بيانات البريد الإلكتروني
- أفضل الممارسات لتنفيذ هذه الميزة في التطبيقات الواقعية

دعونا نتأكد من أن لديك كل ما تحتاجه لمتابعة الأمر بسلاسة.

## دليل التنفيذ

في هذا القسم، سنقوم بتفصيل الخطوات المطلوبة لتوقيع ملف PDF باستخدام رمز الاستجابة السريعة QR الذي يحتوي على كائن بريد إلكتروني.

### تكوين بيانات البريد الإلكتروني وخيارات إشارة رمز الاستجابة السريعة

#### ملخص

نبدأ بإنشاء `Email` كائنٌ يُغلّف جميع التفاصيل الضرورية، مثل العنوان والموضوع والنص. سيتم تشفير هذه البيانات داخل رمز الاستجابة السريعة (QR).

**الخطوة 1: إنشاء كائن بريد إلكتروني**

```csharp
using GroupDocs.Signature.Domain;

// قم بتهيئة كائن البريد الإلكتروني بالخصائص المطلوبة.
Email email = new Email()
{
    Address = "sherlock@holmes.com",
    Subject = "Very important e-mail",
    Body = "Hello, Watson. Reach me ASAP!"
};
```

**توضيح:**
- **عنوان:** عنوان البريد الإلكتروني للمستلم.
- **الموضوع والنص:** حقول قابلة للتخصيص للرسالة.

#### الخطوة 2: تكوين خيارات إشارة رمز الاستجابة السريعة

```csharp
using GroupDocs.Signature.Options;
using System.Drawing;

// قم بإعداد خيارات رمز الاستجابة السريعة (QR code)، وربطها بكائن البريد الإلكتروني الخاص بك.
QrCodeSignOptions options = new QrCodeSignOptions()
{
    EncodeType = QrCodeTypes.QR,
    Data = email,
    HorizontalAlignment = HorizontalAlignment.Left,
    VerticalAlignment = VerticalAlignment.Center,
    Width = 100,
    Height = 100,
    Margin = new Padding(10)
};
```

**توضيح:**
- **نوع الترميز:** يحدد نوع رمز الاستجابة السريعة (QR).
- **بيانات:** يحتوي على كائن البريد الإلكتروني الذي سيتم ترميزه داخل رمز الاستجابة السريعة QR.
- **المحاذاة الأفقية والمحاذاة الرأسية:** التحكم في مكان ظهور رمز الاستجابة السريعة (QR) على الصفحة.

### توقيع الوثيقة وحفظها

بعد تعيين التكوينات، قم بتوقيع المستند باستخدام الخيارات المحددة:

```csharp
using System.IO;

string outputFilePath = "path/to/your/output/document.pdf";

// قم بتوقيع ملف PDF وحفظه في المسار المحدد.
signature.Sign(outputFilePath, options);
```

**توضيح:**
ال `Sign` تطبق الطريقة توقيع رمز الاستجابة السريعة QR الذي تم تكوينه على المستند.
