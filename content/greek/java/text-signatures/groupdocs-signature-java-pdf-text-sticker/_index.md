## Οδηγός Εφαρμογής
Τώρα που είστε έτοιμοι, ας εμβαθύνουμε στην εφαρμογή υπογραφών αυτοκόλλητων κειμένου σε PDF χρησιμοποιώντας το GroupDocs.Signature για Java.

### Υπογραφή εγγράφου με αυτοκόλλητο κειμένου
Αυτή η λειτουργία σάς επιτρέπει να εφαρμόσετε μια οπτικά ελκυστική υπογραφή κειμένου ως αυτοκόλλητο σε ένα έγγραφο PDF. Δείτε πώς:

#### Βήμα 1: Εισαγωγή απαιτούμενων πακέτων
```java
import com.groupdocs.signature.Signature;
import com.groupdocs.signature.domain.enums.TextSignatureImplementation;
import com.groupdocs.signature.domain.enums.PdfTextStickerIcon;
import com.groupdocs.signature.options.appearances.PdfTextStickerAppearance;
import com.groupdocs.signature.options.sign.TextSignOptions;
```

#### Βήμα 2: Ορισμός διαδρομών αρχείων
```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.pdf";
String fileName = Paths.get(filePath).getFileName().toString();
String outputFilePath = new File("YOUR_OUTPUT_DIRECTORY/SignWithTextSticker/").resolve(fileName).toString();
```

#### Βήμα 3: Αρχικοποίηση αντικειμένου υπογραφής
Δημιουργήστε μια παρουσία του `Signature` τάξη, κατευθύνοντάς την στο αρχείο PDF πηγής σας.
```java
Signature signature = new Signature(filePath);
```

#### Βήμα 4: Ρύθμιση παραμέτρων επιλογών υπογραφής κειμένου
Ορίστε επιλογές για την εφαρμογή ενός αυτοκόλλητου κειμένου:
```java
TextSignOptions options = new TextSignOptions("John Smith");
options.setSignatureImplementation(TextSignatureImplementation.Sticker);
```

#### Βήμα 5: Προσαρμόστε την εμφάνιση του αυτοκόλλητου
Προσαρμόστε την εμφάνιση του αυτοκόλλητου κειμένου σας για να το κάνετε να ξεχωρίζει.
```java
PdfTextStickerAppearance appearance = new PdfTextStickerAppearance();
appearance.setIcon(PdfTextStickerIcon.Key); // Επιλέξτε ένα εικονίδιο για οπτική πινελιά
appearance.setOpened(false);
appearance.setContents("Sample");
appearance.setSubject("Sample subject");
appearance.setTitle("Sample Title");

options.setAppearance(appearance);
```

#### Βήμα 6: Ρυθμίσεις ευθυγράμμισης και περιθωρίου
Βεβαιωθείτε ότι η υπογραφή κειμένου σας είναι τοποθετημένη στην κατάλληλη θέση.
```java
options.setVerticalAlignment(VerticalAlignment.Top);
options.setHorizontalAlignment(HorizontalAlignment.Right);
options.setMargin(new Padding(20));
```

#### Βήμα 7: Υπογράψτε το έγγραφο
Εκτελέστε τη διαδικασία υπογραφής και αποθηκεύστε το υπογεγραμμένο έγγραφο.
```java
SignResult signResult = signature.sign(outputFilePath, options);

System.out.println("Source document signed successfully with " + 
                    signResult.getSucceeded().size() + " signature(s).");
System.out.println("File saved at: " + outputFilePath + ".");
```
