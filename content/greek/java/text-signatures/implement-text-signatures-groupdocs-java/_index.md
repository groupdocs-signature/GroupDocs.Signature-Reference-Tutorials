## Ρύθμιση του GroupDocs.Signature για Java

Για να ξεκινήσετε να χρησιμοποιείτε το GroupDocs.Signature στο έργο Java σας, πρέπει να συμπεριλάβετε τη βιβλιοθήκη ως εξάρτηση.

### Εξάρτηση Maven
Προσθέστε τα παρακάτω στο δικό σας `pom.xml` αρχείο:

```xml
<dependency>
    <groupId>com.groupdocs</groupId>
    <artifactId>groupdocs-signature</artifactId>
    <version>23.12</version>
</dependency>
```

### Εξάρτηση Gradle
Για όσους χρησιμοποιούν το Gradle, συμπεριλάβετε αυτό στο `build.gradle` αρχείο:


## Οδηγός Εφαρμογής

Θα αναλύσουμε την εφαρμογή σε σαφή βήματα για να σας βοηθήσουμε να εφαρμόσετε αποτελεσματικά τις υπογραφές κειμένου.

### Δημιουργία υπογραφών κειμένου με εφαρμογή Stamp
#### Επισκόπηση
Ο κύριος στόχος εδώ είναι να προσθέσετε μια υπογραφή κειμένου χρησιμοποιώντας την εφαρμογή Stamp του GroupDocs.Signature, η οποία παρέχει ευελιξία στην τοποθέτηση και τη μορφοποίηση της υπογραφής σε έγγραφα.

#### Ρύθμιση επιλογών υπογραφής
Για να προσαρμόσετε την υπογραφή κειμένου σας, χρησιμοποιήστε `TextSignOptions`:

```java
import com.groupdocs.signature.domain.enums.HorizontalAlignment;
import com.groupdocs.signature.domain.enums.VerticalAlignment;
import com.groupdocs.signature.domain.Padding;
import com.groupdocs.signature.options.sign.TextSignOptions;

// Δημιουργήστε TextSignOptions με το επιθυμητό κείμενο
TextSignOptions options = new TextSignOptions("John Smith");

// Επιλέξτε την εγγενή υλοποίηση για καλύτερη συμβατότητα
options.setSignatureImplementation(TextSignatureImplementation.Native);

// Ευθυγραμμίστε την υπογραφή στην επάνω δεξιά γωνία του εγγράφου
options.setVerticalAlignment(VerticalAlignment.Top);
options.setHorizontalAlignment(HorizontalAlignment.Right);

// Προσθέστε ένα περιθώριο 20 pixel γύρω από το κείμενο
options.setMargin(new Padding(20));
```

#### Υπογραφή και Αποθήκευση
Τέλος, εφαρμόστε την υπογραφή στο έγγραφό σας:

```java
import com.groupdocs.signature.SignResult;

String outputFilePath = "YOUR_OUTPUT_DIRECTORY/SignedWithTextStamp/document.ext";
SignResult signResult = signature.sign(outputFilePath, options);

// Ελέγξτε πόσες υπογραφές εφαρμόστηκαν με επιτυχία
int successfulSignatures = signResult.getSucceeded().size();
```
