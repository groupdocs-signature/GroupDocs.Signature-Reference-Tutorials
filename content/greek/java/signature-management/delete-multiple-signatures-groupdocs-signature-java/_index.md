## Οδηγός Εφαρμογής

### Διαγραφή πολλαπλών υπογραφών σε PDF

Η διαγραφή πολλαπλών υπογραφών από ένα έγγραφο μπορεί να είναι περίπλοκη. Δείτε πώς μπορείτε να την απλοποιήσετε χρησιμοποιώντας το GroupDocs.Signature για Java.

#### Επισκόπηση
Αυτή η ενότητα παρουσιάζει την αναζήτηση και τη διαγραφή διαφόρων τύπων υπογραφών (όπως γραμμωτός κώδικας και κωδικός QR) από ένα έγγραφο.

#### Βήμα 1: Ορισμός διαδρομών
Αρχικά, ορίστε διαδρομές για τα έγγραφα εισόδου και εξόδου.
```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/YourDocument.pdf";
String fileName = Paths.get(filePath).getFileName().toString();
String outputFilePath = new File("YOUR_OUTPUT_DIRECTORY", "DeleteMultipleAdvanced/").resolve(fileName).toString();

// Βεβαιωθείτε ότι ο κατάλογος εξόδου υπάρχει
File dir = new File(outputFilePath.substring(0, outputFilePath.lastIndexOf('/')));
dir.mkdirs();
```
*Γιατί αυτό το βήμα;*Η διασφάλιση της ύπαρξης της διαδρομής εξόδου αποτρέπει τις εξαιρέσεις εισόδου/εξόδου αρχείων.

#### Βήμα 2: Αρχικοποίηση στιγμιαίας παρουσίας υπογραφής
Δημιουργήστε μια παρουσία του `Signature` τάξη για να εργαστείτε με το έγγραφό σας.
```java
signature = new Signature(outputFilePath);
```

#### Βήμα 3: Ορισμός επιλογών αναζήτησης
Ρυθμίστε τις επιλογές αναζήτησης για διαφορετικούς τύπους υπογραφών που θέλετε να διαγράψετε.
```java
BarcodeSearchOptions barcodeOptions = new BarcodeSearchOptions();
QrCodeSearchOptions qrCodeOptions = new QrCodeSearchOptions();

List<SearchOptions> listOptions = Arrays.asList(barcodeOptions, qrCodeOptions);
```

#### Βήμα 4: Αναζήτηση και συλλογή υπογραφών
Χρησιμοποιήστε τις επιλογές αναζήτησης για να βρείτε υπογραφές στο έγγραφό σας.
```java
SearchResult result = signature.search(listOptions);
if (result.getSignatures().size() > 0) {
    List<BaseSignature> signaturesToDelete = new ArrayList<>(result.getSignatures());
} else {
    System.out.println("No signatures were found.");
}
```
*Γιατί να ψάξω;*Ο προσδιορισμός των υπογραφών που θα διαγραφούν είναι κρίσιμος πριν από την αφαίρεσή τους.

#### Βήμα 5: Διαγραφή υπογραφών
Τέλος, προχωρήστε στη διαγραφή των υπογραφών που έχουν συλλεχθεί.
```java
if (!signaturesToDelete.isEmpty()) {
    DeleteResult deleteResult = signature.delete(outputFilePath, signaturesToDelete);
    if (deleteResult.getSucceeded().size() == signaturesToDelete.size()) {
        System.out.println("All signatures were successfully deleted!");
    } else {
        System.out.println("Successfully deleted signatures: " + deleteResult.getSucceeded().size());
        System.out.println("Not deleted signatures: " + deleteResult.getFailed().size());
    }
}
```
*Γιατί αυτό το βήμα;*Επιβεβαιώνει την επιτυχία της διαδικασίας διαγραφής σας.
