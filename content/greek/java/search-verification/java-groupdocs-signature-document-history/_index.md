### Ανάκτηση ιστορικού διεργασίας εγγράφων

#### Επισκόπηση
Αυτή η λειτουργία σάς επιτρέπει να έχετε πρόσβαση σε λεπτομερή αρχεία καταγραφής σχετικά με τις λειτουργίες που εκτελούνται σε ένα έγγραφο. Είναι απαραίτητη για σκοπούς ελέγχου ή για την κατανόηση των τροποποιήσεων που έγιναν κατά τη διαδικασία υπογραφής.

#### Βήματα Υλοποίησης

**Βήμα 1: Ορίστε τη διαδρομή αρχείου σας**
Δημιουργήστε μια παρουσία του `Signature` κλάση καθορίζοντας τη διαδρομή προς το έγγραφό σας:

```java
String filePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_HISTORY";
Signature signature = new Signature(filePath);
```

*Γιατί;*
Αυτό το βήμα αρχικοποιεί μια σύνδεση μεταξύ της εφαρμογής σας και του καθορισμένου εγγράφου, επιτρέποντάς σας να αποκτήσετε πρόσβαση στα μεταδεδομένα και τα αρχεία καταγραφής του.

**Βήμα 2: Ανάκτηση πληροφοριών εγγράφου**
Αποκτήστε πρόσβαση στα αρχεία καταγραφής διεργασιών του εγγράφου ανακτώντας τις πληροφορίες του:

```java
IDocumentInfo documentInfo = signature.getDocumentInfo();
System.out.println("Document Process logs information: count = " + documentInfo.getProcessLogs().size());
```

*Γιατί;*
Η ανάκτηση πληροφοριών εγγράφου είναι ζωτικής σημασίας για την πρόσβαση σε διάφορα μεταδεδομένα, συμπεριλαμβανομένου του αρχείου καταγραφής των διεργασιών που εκτελούνται στο έγγραφο.

**Βήμα 3: Επαναληπτική επεξεργασία αρχείων καταγραφής διεργασιών**
Πραγματοποιήστε επανάληψη σε κάθε αρχείο καταγραφής διεργασιών για να εμφανίσετε τις λεπτομέρειες:

```java
for (ProcessLog processLog : documentInfo.getProcessLogs()) {
    System.out.printf(
        " - operation [%s] on %s. Succeeded/Failed %d/%d. Message: %s%n",
        processLog.getType(),
        processLog.getDate(),
        processLog.getSucceeded(),
        processLog.getFailed(),
        processLog.getMessage()
    );
}
```

*Γιατί;*
Η επανάληψη των αρχείων καταγραφής σάς επιτρέπει να εξαγάγετε και να εμφανίσετε τις λεπτομέρειες κάθε λειτουργίας, όπως τον τύπο, την ημερομηνία, τον αριθμό επιτυχιών ή αποτυχιών και τυχόν σχετικά μηνύματα.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του αρχείου είναι σωστή. Διαφορετικά, `Signature` δεν θα είναι δυνατός ο εντοπισμός του εγγράφου σας.
