## Οδηγός Εφαρμογής

Τώρα που έχουμε έτοιμη την εγκατάστασή μας, ας υλοποιήσουμε τη φόρτωση εγγράφων από έναν διακομιστή FTP χρησιμοποιώντας το GroupDocs.Signature.

### Σύνδεση και ανάκτηση αρχείων από FTP

#### Επισκόπηση
Αυτή η ενότητα εξηγεί πώς να δημιουργήσετε μια σύνδεση με τον διακομιστή FTP σας και να ανακτήσετε αρχεία ως ροές για επεξεργασία σε Java.

#### Βήμα 1: Ρύθμιση σύνδεσης FTP

```java
import org.apache.commons.net.ftp.FTPClient;
import java.io.InputStream;

public class FtpLoader {
    private static InputStream getFileFromFtp(String server, String filePath) throws Exception {
        // Δημιουργήστε μια παρουσία του προγράμματος-πελάτη FTP
        FTPClient client = new FTPClient();

        // Σύνδεση με τον διακομιστή FTP
        client.connect(server);

        // Ανάκτηση αρχείου ως ροή από την καθορισμένη διαδρομή στον διακομιστή FTP
        return client.retrieveFileStream(filePath);
    }
}
```

**Εξήγηση:**
- **FTPClient:** Διευκολύνει τις λειτουργίες FTP χρησιμοποιώντας το Apache Commons Net.
- **retrieveFileStream:** Συνδέεται με τον διακομιστή FTP και ανακτά το αρχείο στη διεύθυνση `filePath` ως ροή εισόδου.

#### Βήμα 2: Φόρτωση εγγράφου στο GroupDocs.Signature

```java
import com.groupdocs.signature.domain.signatures.QrCodeSignature;
import com.groupdocs.signature.options.sign.QrCodeSignOptions;

// Αρχικοποίηση αντικειμένου Signature με το ανακτημένο InputStream
InputStream inputStream = getFileFromFtp("ftp.example.com", "/path/to/document.pdf");
signature.setDocument(inputStream);

// Παράδειγμα προσθήκης υπογραφής κωδικού QR στο έγγραφο
QrCodeSignOptions signOptions = new QrCodeSignOptions("Sample QR Code")
    .setEncodeType(QrCodeTypes.QR)
    .setLeft(100)
    .setTop(100);

signature.sign("signed-document.pdf", signOptions);
```

**Εξήγηση:**
- **Υπογραφή.setDocument:** Ορίζει τη ροή εγγράφων για υπογραφή.
- **Επιλογές Υπογραφής Κωδικού QR:** Ρυθμίζει τις ιδιότητες και τη θέση του κωδικού QR στο έγγραφο.

### Συμβουλές αντιμετώπισης προβλημάτων

- Βεβαιωθείτε ότι τα διαπιστευτήρια και οι διαδρομές του διακομιστή FTP σας είναι σωστά.
