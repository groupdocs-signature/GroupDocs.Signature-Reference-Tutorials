## Οδηγός Εφαρμογής
Ας δούμε την υλοποίηση της φόρτωσης και της υπογραφής ενός εγγράφου χρησιμοποιώντας το GroupDocs.Signature.

### Φόρτωση εγγράφου από τοπικό δίσκο
Η φόρτωση ενός εγγράφου είναι απλή. Ακολουθήστε τα παρακάτω βήματα:

#### 1. Ορισμός διαδρομής αρχείου
Ξεκινήστε καθορίζοντας τη διαδρομή αρχείου όπου είναι αποθηκευμένο το έγγραφό σας.
```java
String filePath = YOUR_DOCUMENT_DIRECTORY + "/YourDocument.docx";
```

#### 2. Εξαγωγή ονόματος αρχείου
Εξαγάγετε το όνομα του αρχείου για χρήση στις διαδρομές εξόδου:
```java
String fileName = new File(filePath).getName();
```

#### 3. Ορισμός διαδρομής εξόδου
Ορίστε τη διαδρομή όπου θα αποθηκευτεί το υπογεγραμμένο έγγραφο.
```java
String outputFilePath = YOUR_OUTPUT_DIRECTORY + "/SignWithText/" + fileName;
```

### Υπογράψτε το Έγγραφο
Στη συνέχεια, θα υπογράψουμε το φορτωμένο έγγραφο χρησιμοποιώντας μια υπογραφή κειμένου.

#### Αρχικοποίηση αντικειμένου υπογραφής
Δημιουργήστε μια παρουσία του `Signature` για τη διαχείριση λειτουργιών αρχείων:
```java
try {
    Signature signature = new Signature(filePath);
```

#### Ρύθμιση επιλογών υπογραφής
Ορίστε τις επιλογές υπογραφής σας. Εδώ, προσθέτουμε μια απλή υπογραφή κειμένου "John Smith":
```java
TextSignOptions options = new TextSignOptions("John Smith");
```

#### Υπογραφή και αποθήκευση εγγράφου
Τέλος, υπογράψτε το έγγραφο και αποθηκεύστε το στην καθορισμένη τοποθεσία.
```java
signature.sign(outputFilePath, options);
```
Εξαιρέσεις εντοπισμού για χειρισμό σφαλμάτων:
```java
catch (GroupDocsSignatureException e) {
    System.err.println("Error signing document: " + e.getMessage());
}
```
