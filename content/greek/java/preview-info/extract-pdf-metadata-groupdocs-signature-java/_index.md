## Οδηγός Εφαρμογής

### Εξαγωγή πληροφοριών εγγράφου

#### Βήμα 1: Αρχικοποίηση αντικειμένου υπογραφής

```java
import com.groupdocs.signature.Signature;

String filePath = "YOUR_DOCUMENT_DIRECTORY/sample.pdf"; // Αντικαταστήστε με την πραγματική διαδρομή PDF
Signature signature = new Signature(filePath);
```
**Εξήγηση**Εδώ, αρχικοποιούμε το `Signature` αντικείμενο παρέχοντάς του τη διαδρομή αρχείου του εγγράφου που θέλετε να αναλύσετε.

#### Βήμα 2: Ανάκτηση πληροφοριών εγγράφου

```java
import com.groupdocs.signature.domain.IDocumentInfo;

IDocumentInfo docInfo = signature.getDocumentInfo();
```
**Εξήγηση**: Το `getDocumentInfo()` Η μέθοδος ανακτά μεταδεδομένα σχετικά με το έγγραφό σας, συμπεριλαμβανομένου του αριθμού σελίδων και του τύπου αρχείου.

#### Βήμα 3: Αριθμός σελίδων εξόδου και τύπος αρχείου

```java
int pageCount = docInfo.getPageCount();
String fileType = docInfo.getFileType().getFileFormat();

System.out.println("Number of Pages: " + pageCount);
System.out.println("File Type: " + fileType);
```
**Εξήγηση**Αυτές οι γραμμές ανακτούν τον συνολικό αριθμό σελίδων και τον τύπο αρχείου εγγράφου, εκτυπώνοντάς τα στην κονσόλα.

#### Βήμα 4: Ανάκτηση διαστάσεων σελίδας

```java
import com.groupdocs.signature.domain.PageInfo;

double maxPageHeight = docInfo.getMaxPageHeight();
double widthForMaxHeight = docInfo.getWidthForMaxHeight();
long fileSizeInBytes = docInfo.getSize();

System.out.println("Maximum Page Height: " + maxPageHeight);
System.out.println("Width for Maximum Height: " + widthForMaxHeight);
System.out.println("File Size in Bytes: " + fileSizeInBytes);

double firstPageWidth = docInfo.getPages().get(0).getWidth();
System.out.println("First Page Width: " + firstPageWidth);
```
**Εξήγηση**Αυτό το απόσπασμα κώδικα εξάγει το μέγιστο ύψος σελίδας, το πλάτος για αυτό το ύψος, το μέγεθος αρχείου και το πλάτος της πρώτης σελίδας.

#### Βήμα 5: Επαναλάβετε την επεξεργασία κάθε σελίδας

```java
for(PageInfo page : docInfo.getPages()){
    int pageNumber = page.getPageNumber();
    double pageHeight = page.getHeight();
    double pageWidth = page.getWidth();

    System.out.println("Page " + pageNumber + ": Height = " + pageHeight + ", Width = " + pageWidth);
}
```
**Εξήγηση**Εδώ, επαναλαμβάνουμε κάθε σελίδα του εγγράφου, ανακτώντας και εκτυπώνοντας το ύψος και το πλάτος της.
