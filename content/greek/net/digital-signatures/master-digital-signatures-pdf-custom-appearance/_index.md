### Ψηφιακές υπογραφές και προσαρμοσμένη εμφάνιση
#### Επισκόπηση
Οι ψηφιακές υπογραφές όχι μόνο επικυρώνουν την αυθεντικότητα των εγγράφων σας, αλλά παρέχουν και ασφάλεια κατά της παραποίησης. Με το GroupDocs.Signature για .NET, μπορείτε να προσαρμόσετε αυτές τις υπογραφές ώστε να ευθυγραμμίζονται με την επωνυμία σας ή τις προσωπικές σας προτιμήσεις.

#### Βήμα προς βήμα εφαρμογή
##### 1. Ρύθμιση επιλογών υπογραφής
Ξεκινήστε ορίζοντας τις επιλογές για την ψηφιακή σας υπογραφή:
```csharp
using System.Drawing;
using GroupDocs.Signature.Options;

DigitalSignOptions options = new DigitalSignOptions("certificate.pfx")
{
    Password = "1234567890",
    Reason = "Approved",
    Contact = "John Smith",
    Location = "New York",
    
    Appearance = new PdfDigitalSignatureAppearance()
    {
        ContactInfoLabel = "C",
        ReasonLabel = "R",
        LocationLabel = "@=>",
        DigitalSignedLabel = "By",
        DateSignedAtLabel = "On",
        Background = Color.FromArgb(50, Color.LightGray),
        FontFamilyName = "Arial",
        FontSize = 8
    },
    
    AllPages = true,
    Width = 160,
    Height = 80,
    VerticalAlignment = VerticalAlignment.Center,
    HorizontalAlignment = HorizontalAlignment.Left,
    Margin = new Padding() { Bottom = 10, Right = 10 },
    
    Border = new Border()
    {
        Visible = true,
        Color = Color.FromArgb(80, Color.DarkGray),
        DashStyle = DashStyle.DashDot,
        Weight = 2
    }
};
```
- **Επεξήγηση παραμέτρων**:
  - `DigitalSignOptions`: Ρυθμίζει την εμφάνιση και τις ιδιότητες της υπογραφής.
  - `Appearance`Επιτρέπει την προσαρμογή οπτικών πτυχών όπως το χρώμα φόντου, οι γραμματοσειρές και οι ετικέτες.

##### 2. Υπογράψτε το Έγγραφο
Χρησιμοποιήστε τις διαμορφωμένες επιλογές για να εφαρμόσετε την ψηφιακή υπογραφή:
```csharp
// Υπογράψτε το έγγραφο με καθορισμένες επιλογές
signature.Sign("signed-output.pdf", options);
```

- Ελέγξτε ξανά τον κωδικό πρόσβασής σας για το πιστοποιητικό, εάν αντιμετωπίσετε προβλήματα πρόσβασης.
