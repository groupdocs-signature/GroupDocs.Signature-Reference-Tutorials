## Οδηγός Εφαρμογής

### Υπογραφή παρουσίασης και αποθήκευση ως διαφορετικό τύπο αρχείου

Προσθέστε ψηφιακές υπογραφές σε παρουσιάσεις και αποθηκεύστε τες σε διαφορετικές μορφές:

#### Δημιουργία Επιλογών Υπογραφής QRCode
Ορίστε τις ιδιότητες της υπογραφής του κωδικού QR χρησιμοποιώντας `QrCodeSignOptions`:

```csharp
using GroupDocs.Signature.Options;

// Ορισμός επιλογών υπογραφής κωδικού QR
QrCodeSignOptions signOptions = new QrCodeSignOptions("JohnSmith")
{
    EncodeType = QrCodeTypes.QR,
    Left = 100, // Οριζόντια θέση στη σελίδα
    Top = 100   // Κάθετη θέση στη σελίδα
};
```

#### Ορισμός επιλογών αποθήκευσης παρουσίασης
Καθορίστε τον τρόπο με τον οποίο θέλετε να αποθηκεύσετε το υπογεγραμμένο έγγραφό σας χρησιμοποιώντας `PresentationSaveOptions`:

```csharp
using GroupDocs.Signature.Domain;

// Ρύθμιση παραμέτρων επιλογών αποθήκευσης για την υπογεγραμμένη παρουσίαση
PresentationSaveOptions saveOptions = new PresentationSaveOptions()
{
    FileFormat = PresentationSaveFileFormat.Tiff,
    OverwriteExistingFiles = true
};
```

#### Υπογραφή και Αποθήκευση
Υπογράψτε το έγγραφό σας και αποθηκεύστε το στην επιθυμητή μορφή:

```csharp
using GroupDocs.Signature;

// Εκτέλεση διαδικασίας υπογραφής και αποθήκευσης
SignResult result = signature.Sign("output/path", signOptions, saveOptions);
```

### Ρύθμιση διαδρομών εγγράφων
Ορίστε σωστά τις διαδρομές για τα αρχεία εισόδου και εξόδου:

```csharp
string sourceDocumentPath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample_Document.docx");
string signedOutputPath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignedDocuments", "Signed_Document.pdf");
```
