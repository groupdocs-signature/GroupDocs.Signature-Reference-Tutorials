## Οδηγός Εφαρμογής
Τώρα που έχετε έτοιμη την εγκατάσταση, ας εφαρμόσουμε την επαλήθευση ψηφιακής υπογραφής. Θα την αναλύσουμε σε διαχειρίσιμα βήματα για να σας διευκολύνουμε.

### Επαλήθευση ψηφιακής υπογραφής
#### Επισκόπηση
Αυτή η λειτουργία δείχνει πώς να επαληθεύσετε την αυθεντικότητα μιας ψηφιακής υπογραφής μέσα σε ένα έγγραφο χρησιμοποιώντας το GroupDocs.Signature για .NET.

#### Βήμα προς βήμα εφαρμογή
1. **Αρχικοποίηση του αντικειμένου υπογραφής**
   Ξεκινήστε δημιουργώντας μια παρουσία του `Signature` τάξη, δείχνοντας το στο υπογεγραμμένο έγγραφό σας:

   ```csharp
   string filePath = "@YOUR_DOCUMENT_DIRECTORY/SAMPLE_SIGNED_MULTI";
   using (Signature signature = new Signature(filePath))
   {
       // Ο κωδικός σας θα μπει εδώ
   }
   ```

2. **Ρύθμιση του DigitalVerifyOptions**
   Διαμορφώστε το `DigitalVerifyOptions` με τα στοιχεία του πιστοποιητικού σας:

   ```csharp
   DigitalVerifyOptions options = new DigitalVerifyOptions("@YOUR_DOCUMENT_DIRECTORY/CertificatePfx")
   {
       Contact = "Mr.Smith",
       Password = "1234567890"
   };
   ```

3. **Επαλήθευση του εγγράφου**
   Εκτελέστε τη διαδικασία επαλήθευσης και ελέγξτε αν είναι επιτυχής:

   ```csharp
   VerificationResult result = signature.Verify(options);

   if (result.IsValid)
   {
       Console.WriteLine($"\nDocument {filePath} was verified successfully!");
       foreach (DigitalSignature item in result.Succeeded)
       {
           Console.WriteLine("\nValid signature is found.");
       }
   }
   else
   {
       Console.WriteLine($"\nDocument {filePath} failed verification process.");
   }
   ```

#### Επεξήγηση παραμέτρων
- **Διαδρομή αρχείου:** Διαδρομή προς το έγγραφο που θέλετε να επαληθεύσετε.
- **Επιλογές Ψηφιακής Επαλήθευσης:** Περιέχει λεπτομέρειες πιστοποιητικού όπως στοιχεία επικοινωνίας και κωδικό πρόσβασης που απαιτούνται για την επικύρωση της υπογραφής.

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι η διαδρομή του αρχείου είναι σωστή και προσβάσιμη.
- Επαληθεύστε την περίοδο ισχύος και τα δικαιώματα του πιστοποιητικού.
