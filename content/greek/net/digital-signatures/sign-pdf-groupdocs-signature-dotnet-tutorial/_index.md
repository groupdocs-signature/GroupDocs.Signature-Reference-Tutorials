Για να ξεκινήσετε με το GroupDocs.Signature για .NET, πρέπει να εγκαταστήσετε τη βιβλιοθήκη στο έργο σας. Ακολουθούν οι μέθοδοι εγκατάστασης:


## Οδηγός Εφαρμογής

### Υπογραφή PDF με υπογραφή κειμένου

Αυτή η ενότητα θα σας καθοδηγήσει στην υπογραφή ενός εγγράφου PDF χρησιμοποιώντας μια υπογραφή κειμένου.

#### Βήμα 1: Ορισμός διαδρομών αρχείων

Αρχικά, ορίστε τις διαδρομές αρχείων εισόδου και εξόδου:

```csharp
string filePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "sample.pdf");
string fileName = Path.GetFileName(filePath);
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignWithText", fileName);
```

#### Βήμα 2: Αρχικοποίηση του αντικειμένου υπογραφής

Δημιουργήστε ένα `Signature` αντικείμενο χρησιμοποιώντας τη διαδρομή αρχείου εισόδου σας:

```csharp
using (Signature signature = new Signature(filePath))
{
    // Περαιτέρω βήματα θα γίνουν εδώ
}
```

#### Βήμα 3: Δημιουργία επιλογών υπογραφής κειμένου

Ρυθμίστε τις επιλογές συμβόλου κειμένου με τις απαραίτητες παραμέτρους όπως η θέση, το μέγεθος και η εμφάνιση:

```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    Αριστερά = 50,
    Top = 200,
    Width = 100,
    Height = 30,
    ForeColor = Color.Red,
    Font = new SignatureFont { Size = 14, FamilyName = "Comic Sans MS" }
};
```

- **Left** και **Κορυφή**: Ορίστε τις συντεταγμένες X και Y για το πού θα εμφανίζεται η υπογραφή.
- **Πλάτος** και **Υψος**: Ορίστε το μέγεθος του πλαισίου κειμένου που περιέχει την υπογραφή σας.
- **Πρόχρωμο**: Καθορίστε το χρώμα του κειμένου.
- **Γραμματοσειρά**: Προσαρμόστε τις ιδιότητες της γραμματοσειράς, συμπεριλαμβανομένου του μεγέθους και της οικογένειας.

#### Βήμα 4: Εφαρμογή της υπογραφής

Τέλος, εφαρμόστε την υπογραφή κειμένου στο PDF και αποθηκεύστε το:

```csharp
SignResult result = signature.Sign(outputFilePath, options);
```
