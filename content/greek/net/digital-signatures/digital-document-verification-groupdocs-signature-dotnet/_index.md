### Χαρακτηριστικό: Ψηφιακή επαλήθευση εγγράφων με χειρισμό εξαιρέσεων

#### Επισκόπηση
Αυτή η λειτουργία επιδεικνύει τη χρήση του GroupDocs.Signature για την επαλήθευση ενός ψηφιακού εγγράφου, ενώ παράλληλα χειρίζεται εξαιρέσεις που ενδέχεται να προκύψουν κατά τη διάρκεια της διαδικασίας.

#### Βήμα προς βήμα εφαρμογή

**1. Ρυθμίστε τις διαδρομές αρχείων σας**
Αντικαταστήστε τα placeholders με τις πραγματικές διαδρομές για τα έγγραφα και τα πιστοποιητικά σας:
```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF";
```

**2. Αρχικοποίηση του GroupDocs.Signature**
Δημιουργήστε ένα `Signature` παράδειγμα για να εργαστείτε με το έγγραφό σας.
```csharp
using (Signature signature = new Signature(filePath))
{
    // Περαιτέρω βήματα εδώ
}
```

**3. Ρύθμιση παραμέτρων του DigitalVerifyOptions**
Ορίστε επιλογές επαλήθευσης, συμπεριλαμβανομένου του καθορισμού της διαδρομής του αρχείου πιστοποιητικού:
```csharp
digitalVerifyOptions options = new digitalVerifyOptions()
{
    CertificateFilePath = "dummy.pfx"
    // Αποσχολιάστε και καθορίστε κωδικό πρόσβασης εάν χρειάζεται: Κωδικός πρόσβασης = "1234567890"
};
```

**4. Εκτελέστε επαλήθευση**
Χρησιμοποιήστε το `Verify` μέθοδος για τον έλεγχο της αυθεντικότητας του εγγράφου.
```csharp
VerificationResult result = signature.Verify(options);
```

**5. Χειρισμός εξαιρέσεων**
Υλοποίηση χειρισμού εξαιρέσεων για συγκεκριμένες εξαιρέσεις GroupDocs.Signature και γενικές εξαιρέσεις συστήματος:
```csharp
catch (GroupDocsSignatureException ex)
{
    Console.WriteLine("GroupDocs Signature Exception: " + ex.Message);
}
catch (Exception ex)
{
    Console.WriteLine("System Exception: " + ex.Message);
}
```
