## Οδηγός Εφαρμογής

### Χαρακτηριστικό: Υπογραφή εγγράφου με κωδικό QR

Αυτή η λειτουργία δείχνει πώς να υπογράψετε ψηφιακά ένα έγγραφο PDF χρησιμοποιώντας έναν κωδικό QR που περιέχει ρυθμίσεις WiFi.

#### Βήμα 1: Προετοιμασία της διαδρομής του εγγράφου και της θέσης εξόδου
```csharp
string filePath = @"YOUR_DOCUMENT_DIRECTORY\SamplePDF.pdf";
string outputFilePath = System.IO.Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "SignedSamplePDF.pdf");
```

#### Βήμα 2: Δημιουργήστε έναν κωδικό QR με πληροφορίες WiFi

Χρησιμοποιώντας το `QrCodeSignOptions`, καθορίστε τις ρυθμίσεις WiFi σας:

```csharp
using GroupDocs.Signature.Options;
using GroupDocs.Signature.Domain;

// Ορίστε τα διαπιστευτήρια WiFi για ενσωμάτωση στον κωδικό QR.
var wifiInfo = new QrCodeWiFi(QrCodeWiFiFrequancy.FiveHundredMhz, "YourNetworkSSID", "password");

QrCodeSignOptions options = new QrCodeSignOptions(wifiInfo)
{
    EncodeType = QrCodeTypes.QR,
    Left = 100,
    Top = 100
};
```

#### Βήμα 3: Υπογράψτε το έγγραφο

Επικαλέστε το `Sign` μέθοδος για την εφαρμογή της υπογραφής κωδικού QR:

```csharp
// Υπογράψτε και αποθηκεύστε το έγγραφο.
signature.Sign(outputFilePath, options);
```

### Συμβουλές αντιμετώπισης προβλημάτων:
- Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε τις ρυθμίσεις WiFi (SSID και κωδικός πρόσβασης) για σωστή κωδικοποίηση.
