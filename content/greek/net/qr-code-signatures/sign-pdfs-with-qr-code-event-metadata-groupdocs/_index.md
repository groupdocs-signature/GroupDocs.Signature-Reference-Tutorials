## Οδηγός Εφαρμογής

Τώρα, ας αναλύσουμε την υλοποίηση σε λογικά μέρη.

### Υπογραφή εγγράφου με κωδικό QR που περιέχει αντικείμενο συμβάντος

Αυτή η λειτουργία επιτρέπει την ενσωμάτωση λεπτομερειών συμβάντος μέσα σε έναν κωδικό QR στα υπογεγραμμένα έγγραφα PDF σας. Βελτιώνει την ακεραιότητα των δεδομένων και παρέχει γρήγορη πρόσβαση σε πρόσθετα μεταδεδομένα χωρίς να επιβαρύνει το έγγραφο.

#### Βήμα 1: Ορισμός του αντικειμένου συμβάντος
Δημιουργήστε ένα `Event` αντιταχθείτε στη διατήρηση πληροφοριών κωδικοποιημένων στον κωδικό QR.
```csharp
// Δημιουργήστε ένα αντικείμενο Event με τις απαραίτητες λεπτομέρειες
Event evnt = new Event()
{
    Title = "GTM(9-00)",
    Description = "General Team Meeting",
    Location = "Conference-Room",
    StartDate = DateTime.Now.Date.AddDays(1).AddHours(9),
    EndDate = DateTime.Now.Date.AddDays(1).AddHours(9).AddMinutes(30)
};
```
*Εξήγηση*Ορίζουμε ένα συμβάν με τίτλο, περιγραφή, τοποθεσία και χρονισμό. Αυτό το αντικείμενο θα κωδικοποιηθεί στον κώδικα QR.

#### Βήμα 2: Ρύθμιση επιλογών υπογραφής κώδικα QR
Ρυθμίστε την εμφάνιση και τα δεδομένα του κωδικού QR.
```csharp
QrCodeSignOptions options = new QrCodeSignOptions
{
    EncodeType = QrCodeTypes.QR,
    Data = evnt, // Αντιστοίχιση του αντικειμένου Event σε ιδιότητα δεδομένων κωδικού QR
    HorizontalAlignment = HorizontalAlignment.Left,
    VerticalAlignment = VerticalAlignment.Center,
    Width = 100,
    Height = 100,
    Margin = new Padding(10)
};
```
*Εξήγηση*Εδώ, ορίζουμε ιδιότητες όπως τον τύπο κωδικοποίησης, την στοίχιση, το μέγεθος και το περιθώριο για τον κώδικα QR.

#### Βήμα 3: Υπογράψτε το έγγραφο
Εφαρμόστε τις επιλογές υπογραφής στο έγγραφό σας.
```csharp
// Ορισμός διαδρομής εξόδου για υπογεγραμμένο έγγραφο
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignWithQRCodeEventObject.pdf");

signature.Sign(outputFilePath, options);
```
*Εξήγηση*: Το `Signature` Το αντικείμενο εφαρμόζει τον διαμορφωμένο κωδικό QR στο PDF και το αποθηκεύει ως νέο αρχείο.
