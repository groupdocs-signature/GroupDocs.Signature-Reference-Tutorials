## Οδηγός Εφαρμογής
### Αναζήτηση υπογραφών γραμμωτού κώδικα
Θα επικεντρωθούμε στη λειτουργία αναζήτησης υπογραφών γραμμωτού κώδικα σε ένα έγγραφο χρησιμοποιώντας το GroupDocs.Signature.

#### Βήμα 1: Ορίστε τη διαδρομή του εγγράφου σας
Ξεκινήστε καθορίζοντας τη διαδρομή προς το έγγραφο-στόχο σας. Εδώ θα αναζητήσει το GroupDocs.Signature γραμμωτούς κώδικες.

```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Δημιουργήστε μια παρουσία υπογραφής
Αρχικοποίηση του `Signature` κλάση με τη διαδρομή αρχείου σας:

```csharp
using (Signature signature = new Signature(filePath))
{
    // Οι λειτουργίες αναζήτησης γραμμωτού κώδικα βρίσκονται εδώ.
}
```

#### Βήμα 3: Αναζήτηση υπογραφών γραμμωτού κώδικα
Χρησιμοποιήστε το `Search<BarcodeSignature>` μέθοδος για την εύρεση γραμμωτών κωδικών στο έγγραφό σας. Αυτό επιστρέφει μια λίστα με τις υπογραφές που βρέθηκαν.

```csharp
List<BarcodeSignature> signatures = signature.Search<BarcodeSignature>(SignatureType.Barcode);
```

#### Βήμα 4: Επαναλάβετε τις υπογραφές που βρέθηκαν
Αναζητήστε ξανά κάθε γραμμωτό κώδικα που βρέθηκε και εμφανίστε τις λεπτομέρειες του:

```csharp
foreach (var barcodeSignature in signatures)
{
    Console.WriteLine($"Found Barcode at page {barcodeSignature.PageNumber} with type {barcodeSignature.EncodeType.TypeName} and text {barcodeSignature.Text}");
}
```

### Επεξήγηση παραμέτρων
- **`filePath`**: Η διαδρομή προς το έγγραφο που θέλετε να αναζητήσετε.
- **`Search<BarcodeSignature>`**: Αναζητά συγκεκριμένα υπογραφές γραμμωτού κώδικα μέσα στο έγγραφο.
- **`PageNumber`, `EncodeType`, `Text`**: Χαρακτηριστικά που παρέχουν πληροφορίες σχετικά με κάθε υπογραφή που βρέθηκε.
