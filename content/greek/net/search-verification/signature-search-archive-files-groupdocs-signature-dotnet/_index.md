# Αποτελεσματική αναζήτηση υπογραφών σε αρχεία αρχειοθέτησης χρησιμοποιώντας το GroupDocs.Signature για .NET

Η αναζήτηση αυτών των υπογραφών σε συμπιεσμένα αρχεία όπως ZIP, 7Z ή TAR μπορεί να είναι δύσκολη χωρίς τα κατάλληλα εργαλεία. Αυτό το σεμινάριο θα σας καθοδηγήσει στο πώς να βελτιστοποιήσετε αυτήν τη διαδικασία χρησιμοποιώντας το GroupDocs.Signature για .NET.

**Τι θα μάθετε:**
- Πώς να ρυθμίσετε το GroupDocs.Signature για .NET
- Αναζήτηση για υπογραφές γραμμωτού κώδικα και κωδικού QR σε αρχεία αρχειοθέτησης
- Διαχείριση αποτελεσμάτων αναζήτησης, συμπεριλαμβανομένων επιτυχημένων και αποτυχημένων διαδικασιών εγγράφων

Ας ξεκινήσουμε με τις απαραίτητες προϋποθέσεις πριν ασχοληθούμε με αυτό το ισχυρό χαρακτηριστικό!

## Οδηγός Εφαρμογής

### Αναζήτηση υπογραφών σε αρχεία εγγράφων

Αυτή η λειτουργία σάς επιτρέπει να αναζητάτε αποτελεσματικά υπογραφές γραμμωτού κώδικα και κωδικού QR σε αρχεία αρχειοθέτησης.

#### Επισκόπηση

Αρχικοποίηση ενός `Signature` αντικειμένου με τη διαδρομή αρχείου ενός αρχειοθετημένου εγγράφου και χρησιμοποιήστε τις επιλογές αναζήτησης για να εντοπίσετε συγκεκριμένους τύπους υπογραφών.

#### Βήμα 1: Αρχικοποίηση αντικειμένου υπογραφής
Δημιουργήστε ένα `Signature` για παράδειγμα, εισάγοντας τη διαδρομή προς το αρχείο σας:

```csharp
string filePath = @"YOUR_DOCUMENT_DIRECTORY\SampleSignedZip.zip";
using (Signature signature = new Signature(filePath))
{
    // Περαιτέρω εφαρμογή...
}
```
**Γιατί:** Ο `Signature` Το αντικείμενο ενσωματώνει όλες τις λειτουργίες για την αναζήτηση και τη διαχείριση υπογραφών μέσα σε έγγραφα.

#### Βήμα 2: Ρύθμιση παραμέτρων επιλογών αναζήτησης
Ορίστε τους τύπους υπογραφών που θέλετε να αναζητήσετε χρησιμοποιώντας συγκεκριμένες επιλογές:

```csharp
BarcodeSearchOptions barcodeOptions = new BarcodeSearchOptions(BarcodeTypes.Code128);
QrCodeSearchOptions qrCodeOptions = new QrCodeSearchOptions(QrCodeTypes.QR);

List<SearchOptions> searchOptionsList = new List<SearchOptions>() { barcodeOptions, qrCodeOptions };
```
**Γιατί:** Ο ορισμός συγκεκριμένων επιλογών βοηθά στον περιορισμό της αναζήτησης σε σχετικούς τύπους υπογραφών, βελτιστοποιώντας την απόδοση.

#### Βήμα 3: Εκτέλεση αναζήτησης
Χρησιμοποιήστε το `Signature.Search` μέθοδος για να βρείτε υπογραφές στο αρχείο σας:

```csharp
SearchResult result = signature.Search(searchOptionsList);
```
**Γιατί:** Αυτή η μέθοδος επεξεργάζεται το/τα έγγραφο/α και επιστρέφει ένα ολοκληρωμένο αποτέλεσμα όλων των υπογραφών που βρέθηκαν.

#### Βήμα 4: Επεξεργασία αποτελεσμάτων
Επαναλάβετε τα αποτελέσματα για να εμφανίσετε ή να καταγράψετε επιτυχημένες ανιχνεύσεις και να διαχειριστείτε τυχόν σφάλματα που εντοπίστηκαν:

```csharp
int documentNumber = 1;
foreach (DocumentResultSignature document in result.Succeeded)
{
    Console.WriteLine($"Document #{documentNumber++}: {document.FileName}. Processed: {document.ProcessingTime}, mls");
    foreach (BaseSignature temp in document.Succeeded)
    {
        Console.WriteLine($"\t\t#{temp.SignatureId}: {temp.SignatureType}");
    }
}

if (result.Failed.Count > 0)
{
    documentNumber = 1;
    foreach (DocumentResultSignature document in result.Failed)
    {
        Console.WriteLine($"ERROR in Document #{documentNumber++}-{document.FileName}: {document.ErrorMessage}, mls");
    }
}
```
**Γιατί:** Η επεξεργασία των αποτελεσμάτων σάς επιτρέπει να κατανοήσετε ποια έγγραφα αναλύθηκαν με επιτυχία και να εντοπίσετε τυχόν προβλήματα που αντιμετώπισαν.
