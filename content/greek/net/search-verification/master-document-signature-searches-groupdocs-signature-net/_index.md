## Οδηγός Εφαρμογής

### Αναζήτηση υπογραφής κειμένου

#### Επισκόπηση

Αυτή η λειτουργία σάς επιτρέπει να αναζητάτε υπογραφές που βασίζονται σε κείμενο σε όλες τις σελίδες ενός εγγράφου, διευκολύνοντας την επαλήθευση της παρουσίας και της θέσης του υπογεγραμμένου περιεχομένου.

#### Βήμα προς βήμα εφαρμογή

1. **Ορισμός επιλογών αναζήτησης**
   
   Διαμορφώστε τις επιλογές για αναζήτηση υπογραφών κειμένου σε όλες τις σελίδες:
   
   ```csharp
   using GroupDocs.Signature.Options;
   
   TextSearchOptions textOptions = new TextSearchOptions() { AllPages = true };
   ```

2. **Εκτελέστε την αναζήτηση**
   
   Χρησιμοποιήστε αυτές τις επιλογές για να εκτελέσετε αναζήτηση υπογραφής κειμένου:
   
   ```csharp
   SearchResult result = signature.Search(textOptions);
   ```

3. **Αποτελέσματα Διαδικασίας**
   
   Επαναλάβετε τις υπογραφές που βρέθηκαν και εμφανίστε λεπτομέρειες:
   
   ```csharp
   if (result.Signatures.Count > 0) {
       foreach (var resSignature in result.Signatures) {
           Console.WriteLine($"Text signature found at page {resSignature.PageNumber} with type {resSignature.SignatureType} and Id#: {resSignature.SignatureId}");
       }
   } else {
       Console.WriteLine("No text signatures were found.");
   }
   ```

### Αναζήτηση Ψηφιακής Υπογραφής

#### Επισκόπηση

Όπως και με τις αναζητήσεις κειμένου, αυτή η λειτουργία σάς επιτρέπει να εντοπίζετε ψηφιακές υπογραφές σε όλο το έγγραφό σας.

#### Βήμα προς βήμα εφαρμογή

1. **Ορισμός επιλογών αναζήτησης**
   
   Ορίστε επιλογές για μια ολοκληρωμένη αναζήτηση:
   
   ```csharp
   using GroupDocs.Signature.Options;
   
   DigitalSearchOptions digitalOptions = new DigitalSearchOptions() { AllPages = true };
   ```

2. **Εκτελέστε την αναζήτηση**
   
   Εκτελέστε την αναζήτηση με τις καθορισμένες επιλογές σας:
   
   ```csharp
   SearchResult result = signature.Search(digitalOptions);
   ```

3. **Αποτελέσματα Διαδικασίας**
   
   Λεπτομέρειες εξόδου για τυχόν υπογραφές που βρέθηκαν:
   
   ```csharp
   if (result.Signatures.Count > 0) {
       foreach (var resSignature in result.Signatures) {
           Console.WriteLine($"Digital signature found at page {resSignature.PageNumber} with type {resSignature.SignatureType} and Id#: {resSignature.SignatureId}");
       }
   } else {
       Console.WriteLine("No digital signatures were found.");
   }
   ```
