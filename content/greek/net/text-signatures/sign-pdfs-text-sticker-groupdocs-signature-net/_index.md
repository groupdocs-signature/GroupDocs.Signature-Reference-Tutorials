## Οδηγός Εφαρμογής
### Υπογραφή εγγράφου PDF με αυτοκόλλητο κειμένου
Αυτή η ενότητα παρουσιάζει τον τρόπο προσθήκης μιας υπογραφής αυτοκόλλητου κειμένου σε ένα έγγραφο PDF χρησιμοποιώντας το GroupDocs.Signature για .NET. Η διαδικασία περιλαμβάνει τον ορισμό επιλογών υπογραφής και τη διαμόρφωση των ρυθμίσεων εμφάνισης.

#### Βήμα 1: Ρύθμιση διαδρομών αρχείων
Ορίστε τις διαδρομές εισόδου και εξόδου για τα αρχεία PDF:
```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY/SAMPLE_PDF";
string outputFilePath = "YOUR_OUTPUT_DIRECTORY/PdfSticker.pdf";
```

#### Βήμα 2: Αρχικοποίηση αντικειμένου υπογραφής
Δημιουργήστε ένα `Signature` αντικείμενο με τη διαδρομή προς το έγγραφο εισαγωγής σας.
```csharp
using (Signature signature = new Signature(filePath))
{
    // Ο κωδικός για την υπογραφή θα τοποθετηθεί εδώ
}
```

#### Βήμα 3: Ρύθμιση παραμέτρων επιλογών υπογραφής κειμένου
Ορίστε και διαμορφώστε επιλογές κειμένου για το αυτοκόλλητό σας:
```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    Left = 50,
    Top = 200,
    Width = 100,
    Height = 30,
    
    SignatureImplementation = TextSignatureImplementation.Sticker,
    
    Appearance = new PdfTextStickerAppearance()
    {
        Icon = PdfTextStickerIcon.Star,
        Opened = false,
        Contents = "Sample",
        Subject = "Sample subject",
        Title = "Sample Title"
    },
    
    Margin = new Padding() { Bottom = 20, Right = 20 },
    ForeColor = Color.Red,
    Font = new SignatureFont { Size = 12, FamilyName = "Comic Sans MS" }
};
```
**Εξήγηση**: Εδώ, ορίζουμε τη θέση (`Left`, `Top`), μέγεθος (`Width`, `Height`), και την εμφάνιση του αυτοκόλλητου. Το `SignatureImplementation` Η ιδιότητα καθορίζει ότι πρόκειται για υπογραφή αυτοκόλλητου κειμένου.

#### Βήμα 4: Εκτέλεση υπογραφής
Καλέστε το `Sign` μέθοδος για την εφαρμογή της υπογραφής:
```csharp
SignResult signResult = signature.Sign(outputFilePath, options);
```
Αυτή η μέθοδος θα αποθηκεύσει το υπογεγραμμένο έγγραφό σας στην καθορισμένη διαδρομή εξόδου.
