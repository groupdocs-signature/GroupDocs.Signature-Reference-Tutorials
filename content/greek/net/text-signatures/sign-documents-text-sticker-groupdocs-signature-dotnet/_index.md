## Οδηγός Εφαρμογής

Αυτή η ενότητα σας καθοδηγεί στην εφαρμογή μιας υπογραφής αυτοκόλλητου κειμένου.

### Επισκόπηση

Η λειτουργία επιτρέπει την τοποθέτηση ενός "αυτοκόλλητου" κειμένου πάνω από έγγραφα, ιδανικό για ψηφιακές υπογραφές που απαιτούν οπτική αναπαράσταση και μεταδεδομένα.

### Βήμα προς βήμα εφαρμογή

#### 1. Ορισμός διαδρομών εγγράφων
Ορίστε τις διαδρομές των αρχείων σας:
```csharp
string filePath = @"YOUR_DOCUMENT_DIRECTORY/Sample.pdf"; // Αντικατάσταση με την πραγματική διαδρομή
string fileName = System.IO.Path.GetFileName(filePath);
string outputFilePath = System.IO.Path.Combine(@"YOUR_OUTPUT_DIRECTORY", "SignWithTextSticker", fileName);
```

#### 2. Δημιουργία επιλογών υπογραφής κειμένου
Διαμορφώστε τις επιλογές πινακίδας κειμένου για το αυτοκόλλητο:
```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    Υλοποίηση Υπογραφής = TextSignatureImplementation.Sticker,
    
    Appearance = new PdfTextStickerAppearance()
    {
        Icon = PdfTextStickerIcon.Key,
        Opened = false,
        Contents = "Sample",
        Subject = "Sample subject",
        Title = "Sample Title"
    },

    VerticalAlignment = VerticalAlignment.Center,
    HorizontalAlignment = HorizontalAlignment.Left,
    Margin = new Padding(20)
};
```
- **SignatureImplementation**: Ορισμός σε `Sticker` για τη λειτουργία αυτοκόλλητου.
- **Ιδιότητες εμφάνισης**: Προσαρμόστε οπτικά στοιχεία όπως εικονίδια και μεταδεδομένα.

#### 3. Υπογράψτε το Έγγραφο
Εκτελέστε τη διαδικασία υπογραφής:
```csharp
using (Signature signature = new Signature(filePath))
{
    SignResult signResult = signature.Sign(outputFilePath, options);
}
```

### Συμβουλές αντιμετώπισης προβλημάτων
- Βεβαιωθείτε ότι οι διαδρομές αρχείων είναι σωστές για να αποφύγετε `FileNotFoundException`.
- Επαληθεύστε ότι η άδειά σας είναι έγκυρη και έχει εφαρμοστεί σωστά για προηγμένες λειτουργίες.
