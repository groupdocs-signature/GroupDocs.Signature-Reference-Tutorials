## Οδηγός Εφαρμογής

### Επισκόπηση λειτουργιών: Υπογραφή κειμένου με ακτινωτό πινέλο διαβάθμισης
Αυτή η λειτουργία σάς επιτρέπει να υπογράφετε έγγραφα με έναν αισθητικά ευχάριστο τρόπο, εφαρμόζοντας ένα ακτινωτό πινέλο διαβάθμισης. Ας αναλύσουμε τη διαδικασία υλοποίησης:

#### Βήμα 1: Ορίστε τις διαδρομές των εγγράφων σας
Αρχικά, ορίστε τις διαδρομές για τα αρχεία εισόδου και εξόδου:
```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY\Sample.pdf";
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignWithBrushes", "SignedLinearRadialBrush.pdf");
```

#### Βήμα 2: Αρχικοποίηση αντικειμένου υπογραφής
Δημιουργήστε ένα `Signature` παράδειγμα με τη διαδρομή προς το PDF σας:
```csharp
using (Signature signature = new Signature(filePath))
{
    // Περαιτέρω βήματα θα εκτελεστούν εντός αυτού του μπλοκ
}
```

#### Βήμα 3: Ρύθμιση παραμέτρων του TextSignOptions
Ρυθμίστε τις επιλογές για την εφαρμογή της υπογραφής κειμένου, συμπεριλαμβανομένων των ρυθμίσεων φόντου και στοίχισης:
```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    Φόντο = new Background()
    {
        Color = Color.LimeGreen,
        Transparency = 0.5,
        Brush = new RadialGradientBrush(Color.LimeGreen, Color.DarkGreen)
    },
    Width = 100,
    Height = 80,
    VerticalAlignment = Domain.VerticalAlignment.Center,
    HorizontalAlignment = Domain.HorizontalAlignment.Center,
    Margin = new Padding() { Top = 20, Right = 20 },
    SignatureImplementation = TextSignatureImplementation.Image
};
```
- **Background**: Προσαρμογή χρησιμοποιώντας `RadialGradientBrush` για ομαλή μετάβαση μεταξύ των χρωμάτων.
- **Διαστάσεις & Ευθυγράμμιση**: Ορίστε το μέγεθος και τη θέση της υπογραφής κειμένου σας.

#### Βήμα 4: Υπογραφή και αποθήκευση του εγγράφου
Εφαρμόστε τις διαμορφωμένες επιλογές υπογραφής για να υπογράψετε το έγγραφο:
```csharp
SignResult signResult = signature.Sign(outputFilePath, options);
```
