## Οδηγός Εφαρμογής

Ας αναλύσουμε βήμα προς βήμα τον τρόπο εφαρμογής υπογραφών μεταδεδομένων.

### Επισκόπηση των υπογραφών μεταδεδομένων

Οι υπογραφές μεταδεδομένων ενσωματώνουν βασικές πληροφορίες απευθείας σε έγγραφα χωρίς να αλλοιώνουν το περιεχόμενό τους. Αυτή η μέθοδος είναι ιδανική για την παρακολούθηση της προέλευσης των εγγράφων, της συγγραφής τους και άλλων.

#### Βήμα 1: Προετοιμάστε το έγγραφό σας

Αρχικά, βεβαιωθείτε ότι έχετε έτοιμη τη διαδρομή προς το έγγραφο του Word:
```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY";
```

#### Βήμα 2: Ρύθμιση παραμέτρων υπογραφών μεταδεδομένων

Δημιουργήστε ένα `MetadataSignOptions` αντικείμενο και προσθέστε διάφορες καταχωρίσεις μεταδεδομένων. Κάθε καταχώρηση αποτελείται από ένα κλειδί (π.χ., Συγγραφέας) και την αντίστοιχη τιμή του.

```csharp
// Επιλογή δημιουργίας μεταδεδομένων με προκαθορισμένο κείμενο μεταδεδομένων
MetadataSignOptions options = new MetadataSignOptions();

// Προσθήκη υπογραφών μεταδεδομένων
options.Add(new WordProcessingMetadataSignature("Author", "Mr.Sherlock Holmes"));
options.Add(new WordProcessingMetadataSignature("CreatedOn", DateTime.Now));
options.Add(new WordProcessingMetadataSignature("DocumentId", 123456));
options.Add(new WordProcessingMetadataSignature("SignatureId", 123.456D));
options.Add(new WordProcessingMetadataSignature("Amount", 123.456M));
options.Add(new WordProcessingMetadataSignature("Total", 123.456F));
```

#### Βήμα 3: Υπογράψτε το έγγραφο

Επικαλέστε το `Sign` μέθοδος για την εφαρμογή υπογραφών μεταδεδομένων και την αποθήκευση του υπογεγραμμένου εγγράφου.

```csharp
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignedWithMetadata.docx");

// Υπογράψτε το έγγραφο
SignResult result = signature.Sign(outputFilePath, options);

// Σχόλια κονσόλας (προαιρετικά)
Console.WriteLine($"\nSource document signed successfully with {result.Succeeded.Count} signature(s).\nFile saved at {outputFilePath}.");
```

- **Μη έγκυρες διαδρομές**Βεβαιωθείτε ότι οι διαδρομές των αρχείων σας είναι σωστές για να αποφύγετε `FileNotFoundException`.
