## Οδηγός Εφαρμογής

### Επισκόπηση της αναζήτησης υπογραφών μεταδεδομένων
Η αναζήτηση υπογραφών μεταδεδομένων σε ένα PDF σάς επιτρέπει να ανακτήσετε και να χειριστείτε κρίσιμα δεδομένα που είναι ενσωματωμένα στο έγγραφο. Ακολουθήστε τα παρακάτω βήματα για να εφαρμόσετε αυτήν τη λειτουργικότητα.

#### Βήμα 1: Αρχικοποίηση του `Signature` Αντικείμενο
Ξεκινήστε δημιουργώντας μια παρουσία του `Signature` κλάση, παρέχοντάς της τη διαδρομή προς το αρχείο PDF σας:

```csharp
using (Signature signature = new Signature(filePath))
{
    // Θα ακολουθήσει επιπλέον κώδικας εδώ
}
```

Αυτό το αντικείμενο χρησιμεύει ως πύλη για την αναζήτηση και τη διαχείριση υπογραφών μέσα στο έγγραφό σας.

#### Βήμα 2: Αναζήτηση υπογραφών μεταδεδομένων
Χρησιμοποιήστε το `Search` μέθοδος με `PdfMetadataSignature` για να εντοπίσετε όλες τις καταχωρίσεις μεταδεδομένων στο αρχείο PDF:

```csharp
List<PdfMetadataSignature> signatures = signature.Search<PdfMetadataSignature>(SignatureType.Metadata);
```

Αυτή η γραμμή ανακτά μια λίστα υπογραφών μεταδεδομένων, επιτρέποντας περαιτέρω λειτουργίες.

#### Βήμα 3: Ανάκτηση και εμφάνιση τιμών μεταδεδομένων
Επαναλάβετε κάθε `PdfMetadataSignature` για πρόσβαση σε συγκεκριμένες καταχωρήσεις όπως Συγγραφέας, ΔημιουργήθηκεΣτις, κ.λπ. Παρακάτω παρατίθενται παραδείγματα για την ανάκτηση διαφόρων τύπων δεδομένων:

```csharp
// Παράδειγμα για την ανάκτηση της υπογραφής «Συγγραφέας» ως συμβολοσειρά
PdfMetadataSignature mdSignature = signatures.FirstOrDefault(p => p.Name == "Author");
Console.WriteLine($"[{mdSignature.Name}] as String = {mdSignature.ToString()}");
```

Συνεχίστε με παρόμοιο τρόπο για να εξαγάγετε άλλες τιμές μεταδεδομένων, μετατρέποντάς τες στους αντίστοιχους τύπους τους, όπως ημερομηνία, ακέραιος, διπλός, κ.λπ.

```csharp
// Παράδειγμα για την ανάκτηση της υπογραφής 'CreatedOn' ως ημερομηνίας
mdSignature = signatures.FirstOrDefault(p => p.Name == "CreatedOn");
Console.WriteLine($"[{mdSignature.Name}] as Date = {mdSignature.ToDateTime().ToShortDateString()}");
```

Χειριστείτε εξαιρέσεις για να διασφαλίσετε ότι η εφαρμογή σας παραμένει ισχυρή:

```csharp
catch (Exception ex)
{
    Console.WriteLine($"Error obtaining signature: {ex.Message}");
}
```
