## Οδηγός Εφαρμογής

Θα αναλύσουμε την υλοποίηση σε λογικά βήματα για την υπογραφή ενός υπολογιστικού φύλλου Excel χρησιμοποιώντας υπογραφές μεταδεδομένων.

### Βήμα 1: Ορισμός υπογραφών μεταδεδομένων

Δημιουργήστε μια λίστα με καταχωρίσεις μεταδεδομένων που θα προστεθούν στο έγγραφό σας. Κάθε καταχώρηση θα πρέπει να έχει συγκεκριμένους τύπους δεδομένων και τιμές που σχετίζονται με τις ανάγκες σας.

```csharp
using GroupDocs.Signature.Domain;
using System;

// Δημιουργήστε επιλογές υπογραφής μεταδεδομένων για να καθορίσετε υπογραφές μεταδεδομένων
MetadataSignOptions options = new MetadataSignOptions();

SpreadsheetMetadataSignature[] signatures = new SpreadsheetMetadataSignature[]
{
    new SpreadsheetMetadataSignature("Author", "Mr. Scherlock Holmes"), // Προσθήκη συγγραφέα ως τιμή συμβολοσειράς
    new SpreadsheetMetadataSignature("CreatedOn", DateTime.Now), // Προσθήκη ημερομηνίας δημιουργίας με την τρέχουσα χρονική σήμανση
    new SpreadsheetMetadataSignature("DocumentId", 123456), // Αντιστοίχιση ακέραιου αριθμού αναγνωριστικού εγγράφου
    new SpreadsheetMetadataSignature("SignatureId", 123.456D), // Αντιστοίχιση διπλού αναγνωριστικού υπογραφής
    new SpreadsheetMetadataSignature("Amount", 123.456M), // Ορίστε το Ποσό ως δεκαδική τιμή
    new SpreadsheetMetadataSignature("Total", 123.456F) // Ορισμός συνόλου με τιμή κινητής υποδιαστολής
};

options.Signatures.AddRange(signatures); // Προσθήκη όλων των υπογραφών μεταδεδομένων στις επιλογές
```

### Βήμα 2: Υπογραφή και αποθήκευση του εγγράφου

Με τις επιλογές μεταδεδομένων που έχουν ρυθμιστεί, μπορείτε πλέον να υπογράψετε το έγγραφό σας και να το αποθηκεύσετε.

```csharp
using GroupDocs.Signature.Options;

string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignSpreadsheetWithMetadata", "SignedWithMetadata.xlsx");

// Υπογράψτε το έγγραφο και αποθηκεύστε το στην καθορισμένη διαδρομή εξόδου
SignResult result = signature.Sign(outputFilePath, options);
```

### Παράμετροι και τιμές επιστροφής

- **Υπογραφή(διαδρομήαρχείου)**: Αρχικοποιεί μια νέα παρουσία του `Signature` κλάση με τη διαδρομή αρχείου.
- **Επιλογές Υπογραφής Μεταδεδομένων**: Αντιπροσωπεύει τις ρυθμίσεις υπογραφής μεταδεδομένων.
- **Μεταδεδομένα υπολογιστικού φύλλουΥπογραφή(όνομα, τιμή)**: Ορίζει μεμονωμένες καταχωρίσεις μεταδεδομένων.
- **ΥπογραφήΑποτέλεσμα**: Το αντικείμενο αποτελέσματος που περιέχει πληροφορίες σχετικά με τη διαδικασία υπογραφής.
