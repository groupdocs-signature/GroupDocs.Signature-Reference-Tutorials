### Δυνατότητα: Δημιουργία προεπισκόπησης σελίδων εγγράφων
#### Επισκόπηση
Δημιουργήστε εικόνες JPEG από κάθε σελίδα ενός εγγράφου, κάτι χρήσιμο για την προεπισκόπηση μεγάλων εγγράφων ή την κοινή χρήση δειγμάτων σελίδων με πελάτες.

#### Βήματα Υλοποίησης
**Βήμα 1: Αρχικοποίηση του αντικειμένου υπογραφής**
Δημιουργήστε μια παρουσία του `Signature` κλάση, καθορίζοντας τη διαδρομή του αρχείου PDF.

```csharp
string filePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "Sample.pdf");
using (Signature signature = new Signature(filePath))
{
    // Περαιτέρω βήματα θα εφαρμοστούν εδώ
}
```

**Βήμα 2: Ρύθμιση επιλογών προεπισκόπησης**
Ορίστε τον τρόπο αποθήκευσης κάθε προεπισκόπησης σελίδας χρησιμοποιώντας το `PreviewOptions` τάξη.

```csharp
PreviewOptions previewOption = new PreviewOptions(pageStream => 
    Path.Combine("YOUR_OUTPUT_DIRECTORY", "GeneratePreviewFolder", $"image-{pageStream.PageNumber}.jpg")
)
{
    PreviewFormat = PreviewOptions.PreviewFormats.JPEG,
};
```

**Βήμα 3: Διαχείριση ροών σελίδων**
Βεβαιωθείτε ότι τα προσωρινά αρχεία καθαρίζονται μετά τη δημιουργία προεπισκοπήσεων.

```csharp
previewOption.StreamProvider.AfterSavePage += (sender, args) => 
    File.Delete(args.PageStream.FilePath);
```

**Βήμα 4: Δημιουργία προεπισκοπήσεων**
Εκτελέστε τη διαδικασία δημιουργίας προεπισκόπησης με τις διαμορφωμένες επιλογές.

```csharp
signature.GeneratePreview(previewOption);
```

### Δυνατότητα: Δημιουργία και διαχείριση ροής για προεπισκόπηση
#### Επισκόπηση
Η αποτελεσματική διαχείριση ροής είναι ζωτικής σημασίας για τη διασφάλιση της βέλτιστης χρήσης πόρων κατά τη διαδικασία δημιουργίας προεπισκόπησης.

#### Βήματα Υλοποίησης
**Βήμα 1: Δημιουργία ροών σελίδων**
Ορίστε μια μέθοδο για τη δημιουργία ροών για κάθε εικόνα σελίδας, διασφαλίζοντας ότι υπάρχουν εκ των προτέρων κατάλογοι.

```csharp
Stream CreatePageStream(PreviewPageData pageData)
{
    string imageFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "GeneratePreviewFolder", $"image-{pageData.PageNumber}.jpg");
    Directory.CreateDirectory(Path.GetDirectoryName(imageFilePath));
    return new FileStream(imageFilePath, FileMode.Create);
}
```

**Βήμα 2: Απελευθέρωση ροών σελίδας**
Απορρίψτε τις ροές σε ελεύθερους πόρους μετά τη χρήση.

```csharp
void ReleasePageStream(PreviewPageData pageData, Stream pageStream)
{
    pageStream.Dispose();
    string imageFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "GeneratePreviewFolder", $"image-{pageData.PageNumber}.jpg");
}
```
