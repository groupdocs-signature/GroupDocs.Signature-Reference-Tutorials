### Ανάκτηση υποστηριζόμενων μορφών αρχείων

**Επισκόπηση:**
Αυτή η λειτουργία επιτρέπει στην εφαρμογή σας να εμφανίζει δυναμικά όλους τους τύπους αρχείων που υποστηρίζει η βιβλιοθήκη GroupDocs.Signature, διευκολύνοντας την απρόσκοπτη διαχείριση και επεξεργασία διαφόρων εγγράφων.

#### Βήμα 1: Ανάκτηση υποστηριζόμενων τύπων αρχείων

Ξεκινήστε ανακτώντας μια συλλογή από υποστηριζόμενες μορφές αρχείων:

```csharp
IEnumerable<FileType> supportedFileTypes = FileType.GetSupportedFileTypes().OrderBy(f => f.Extension);
```

**Εξήγηση:**
- `FileType.GetSupportedFileTypes()` ανακτά όλους τους υποστηριζόμενους τύπους αρχείων.
- `.OrderBy(f => f.Extension)` Ταξινομεί τη λίστα αλφαβητικά κατά επέκταση αρχείου.

#### Βήμα 2: Εμφάνιση πληροφοριών μορφής αρχείου

Επαναλάβετε κάθε τύπο αρχείου και εξαγάγετε τις λεπτομέρειες του:

```csharp
foreach (FileType fileType in supportedFileTypes)
{
    Console.WriteLine(fileType);
}
```

**Εξήγηση:**
- Αυτός ο βρόχος διασχίζει κάθε `FileType` αντικείμενο, εμφανίζοντας βασικές πληροφορίες όπως η επέκταση και ο τύπος MIME.
