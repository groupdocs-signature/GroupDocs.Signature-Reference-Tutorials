### Επισκόπηση

#### Βήμα 1: Ρύθμιση του έργου σας
Βεβαιωθείτε ότι το περιβάλλον .NET είναι έτοιμο και ότι έχετε εγκαταστήσει το GroupDocs.Signature όπως φαίνεται παραπάνω. 

#### Βήμα 2: Υλοποίηση ανάκτησης ιστορικού διαδικασίας εγγράφων
Δημιουργήστε μια κλάση για τη διαχείριση της ανάκτησης του ιστορικού εγγράφων:

```csharp
using System;
using System.IO;
using GroupDocs.Signature;
using GroupDocs.Signature.Domain;

public class GetDocumentProcessHistoryFeature
{
    public static void Run()
    {
        string filePath = Path.Combine(@"YOUR_DOCUMENT_DIRECTORY", "sample.pdf");
        
        // Αρχικοποίηση της παρουσίας υπογραφής
        using (var signature = new Signature(filePath))
        {
            // Ανάκτηση ιστορικού εγγράφων
            var history = signature.GetHistory();
            
            foreach (var entry in history)
            {
                Console.WriteLine($"Action: {entry.Action}");
                Console.WriteLine($"Date: {entry.DateTime}");
                Console.WriteLine($"User: {entry.UserId}");
                Console.WriteLine();
            }
        }
    }
}
```

**Εξήγηση**: 
- `GetHistory()` Η μέθοδος ανακτά μια λίστα ενεργειών που εκτελούνται στο έγγραφο.
- Κάθε καταχώρηση σε αυτό το ιστορικό περιλαμβάνει λεπτομέρειες όπως τον τύπο ενέργειας, την ημερομηνία και το αναγνωριστικό χρήστη.
