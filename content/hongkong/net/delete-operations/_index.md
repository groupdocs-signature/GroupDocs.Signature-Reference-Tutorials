# GroupDocs.Signature for .NET 中的刪除操作

## 介紹

現代商業環境中的文件管理需要對簽名和其他身份驗證元素進行精確控制。 GroupDocs.Signature for .NET 提供強大的功能，當文件中不再需要或需要更新時，您可以透過程式設計刪除各種簽章類型。本指南涵蓋 API 中所有可用的刪除操作，並附有詳細的說明和實作範例。
