# วิธีการลงนามเอกสารด้วยตราประทับข้อความโดยใช้ GroupDocs.Signature สำหรับ .NET

## การแนะนำ

คุณกำลังมองหาระบบการลงนามเอกสารอัตโนมัติในแอปพลิเคชันของคุณอยู่หรือเปล่า? ไม่ว่าจะเป็นสัญญา ข้อตกลง หรือเอกสารทางการ สิ่งสำคัญคือต้องมั่นใจว่าเอกสารเหล่านั้นได้รับการลงนามอย่างมีประสิทธิภาพและถูกต้อง บทช่วยสอนนี้จะแนะนำคุณเกี่ยวกับการใช้งาน **GroupDocs.Signature สำหรับ .NET** การลงนามในเอกสารด้วยการประทับตราข้อความ

ในบทความนี้ เราจะเจาะลึกวิธีการนำ GroupDocs.Signature สำหรับ .NET มาใช้เพื่อเพิ่มลายเซ็นข้อความลงในเอกสารของคุณได้อย่างราบรื่นและปลอดภัย เราจะครอบคลุมทุกอย่างตั้งแต่การตั้งค่าสภาพแวดล้อมของคุณไปจนถึงการใช้งานจริงของไลบรารีอันทรงพลังนี้

### สิ่งที่คุณจะได้เรียนรู้:
- วิธีการติดตั้งและตั้งค่า GroupDocs.Signature สำหรับ .NET
- คำแนะนำทีละขั้นตอนในการลงนามเอกสารโดยใช้ตราประทับข้อความ
- ตัวเลือกการกำหนดค่าที่สำคัญและเคล็ดลับการแก้ไขปัญหา
- กรณีการใช้งานในโลกแห่งความเป็นจริงและกลยุทธ์การเพิ่มประสิทธิภาพการทำงาน

มาเจาะลึกข้อกำหนดเบื้องต้นที่คุณต้องปฏิบัติตามกันดีกว่า

## คู่มือการใช้งาน

### ลงนามในเอกสารพร้อมประทับตราข้อความ

มาดูวิธีการลงนามเอกสารโดยใช้ฟีเจอร์ประทับตราข้อความกัน

#### ขั้นตอนที่ 1: เตรียมสภาพแวดล้อม

ตรวจสอบให้แน่ใจว่าโครงการของคุณได้ติดตั้ง GroupDocs.Signature และตั้งค่าตามที่อธิบายไว้ข้างต้น 

#### ขั้นตอนที่ 2: สร้างตัวเลือกลายเซ็น

กำหนดค่า `TextSignOptions` คลาสเพื่อระบุรายละเอียดลายเซ็น เช่น ข้อความ การจัดตำแหน่ง และระยะขอบ:

```csharp
using System;
using GroupDocs.Signature;
using GroupDocs.Signature.Options;

string filePath = "YOUR_DOCUMENT_DIRECTORY"; // เส้นทางไปยังไฟล์ต้นฉบับ
string fileName = System.IO.Path.GetFileName(filePath);
string outputFilePath = System.IO.Path.Combine("YOUR_OUTPUT_DIRECTORY", "SignWithTextStamp", fileName);

using (Signature signature = new Signature(filePath))
{
    TextSignOptions options = new TextSignOptions("John Smith")
    {
        SignatureImplementation = TextSignatureImplementation.Native,
        VerticalAlignment = VerticalAlignment.Center,
        HorizontalAlignment = HorizontalAlignment.Left,
        Margin = new Padding(20)
    };

    SignResult signResult = signature.Sign(outputFilePath, options);
}
```

**พารามิเตอร์ที่อธิบาย:**
- `TextSignOptions`: กำหนดเนื้อหาข้อความและลักษณะของแสตมป์ของคุณ
  - `SignatureImplementation`:ระบุการใช้การใช้งานดั้งเดิมเพื่อประสิทธิภาพที่เหมาะสมที่สุด
  - `VerticalAlignment` - `HorizontalAlignment`:จัดตำแหน่งลายเซ็นให้อยู่ในตำแหน่งที่ต้องการ
  - `Margin`: เพิ่มระยะห่างรอบข้อความเพื่อป้องกันไม่ให้ถูกตัดออก

**เคล็ดลับการแก้ไขปัญหา:**
- ตรวจสอบให้แน่ใจว่าเส้นทางไฟล์ถูกต้อง หากเส้นทางไม่ถูกต้องจะทำให้เกิดข้อผิดพลาด
- ตรวจสอบว่ารูปแบบเอกสารได้รับการสนับสนุนโดย GroupDocs.Signature

### โหลดเอกสารเพื่อลงนาม

ก่อนที่จะลงนามคุณต้องโหลดเอกสารของคุณลงใน `Signature` วัตถุ ทำได้ดังนี้:

```csharp
string filePath = "YOUR_DOCUMENT_DIRECTORY"; // เส้นทางไปยังไฟล์ต้นฉบับ

using (Signature signature = new Signature(filePath))
{
    // เอกสารนี้พร้อมสำหรับการลงนามดำเนินการแล้ว
}
```
