## คู่มือการใช้งาน

มาแบ่งการใช้งานออกเป็นขั้นตอนที่ชัดเจนสำหรับแต่ละฟีเจอร์กัน

### โหลดเอกสารจากดิสก์ภายในเครื่อง

#### ภาพรวม
การโหลดเอกสารเป็นขั้นตอนแรกของการลงนามดิจิทัล กระบวนการนี้เกี่ยวข้องกับการอ่านไฟล์จากดิสก์ภายในเครื่องของคุณและเตรียมเอกสารสำหรับการลงนาม

#### การดำเนินการแบบทีละขั้นตอน

**1. ตั้งค่าเส้นทางไฟล์**
กำหนดเส้นทางสำหรับไฟล์อินพุตและเอาต์พุตของคุณ:
```csharp
string filePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleWordProcessing.docx");
string fileName = Path.GetFileName(filePath);
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "LoadDocumentFromLocalDisk", fileName);
```

**2. เริ่มต้นวัตถุลายเซ็น**
สร้าง `Signature` วัตถุที่มีเส้นทางเอกสาร:
```csharp
using (Signature signature = new Signature(filePath))
{
    // ขั้นตอนต่อไปจะดำเนินการภายในบริบทนี้
}
```

**3. กำหนดตัวเลือกการลงนาม**
ตั้งค่าตัวเลือกว่าคุณต้องการลงนามในเอกสารของคุณอย่างไร:
```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    // กำหนดค่าการตั้งค่าเพิ่มเติม เช่น ตำแหน่งและขนาดที่นี่
};
```

**4. ลงนามในเอกสาร**
ใช้ลายเซ็นและบันทึกผลลัพธ์:
```csharp
SignResult result = signature.Sign(outputFilePath, options);
// วัตถุ 'ผลลัพธ์' เก็บรายละเอียดเกี่ยวกับกระบวนการลงนาม
```

### บันทึกเอกสารที่ลงนาม

#### ภาพรวม
หลังจากลงนามเอกสารแล้ว คุณจะต้องบันทึกเอกสารอย่างปลอดภัยในไดเร็กทอรีเอาต์พุต

#### การดำเนินการแบบทีละขั้นตอน

**1. ตั้งค่าเส้นทางไฟล์**
เช่นเดียวกับก่อนหน้านี้ กำหนดเส้นทางสำหรับอินพุตและเอาต์พุต:
```csharp
string filePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleWordProcessing.docx");
string fileName = Path.GetFileName(filePath);
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "SaveSignedDocument", fileName);
```

**2. เริ่มต้นวัตถุลายเซ็น**
นำกลับมาใช้ใหม่ `Signature` วัตถุที่จะจัดการการลงนาม:
```csharp
using (Signature signature = new Signature(filePath))
{
    // ดำเนินการลงนามที่นี่
}
```

**3. กำหนดและใช้ตัวเลือกการลงนาม**
กำหนดค่าตัวเลือกเครื่องหมายข้อความของคุณตามต้องการ:
```csharp
TextSignOptions options = new TextSignOptions("John Smith")
{
    // กำหนดค่าการกำหนดค่าสำหรับลักษณะที่ปรากฏของลายเซ็น
};
```

**4. บันทึกเอกสาร**
ดำเนินการลงนามและบันทึกไฟล์ไปยังตำแหน่งที่ต้องการ:
```csharp
SignResult result = signature.Sign(outputFilePath, options);
// เอกสารที่ลงนามแล้วจะถูกบันทึกไว้ใน 'outputFilePath'
```

### เคล็ดลับการแก้ไขปัญหา

- ตรวจสอบให้แน่ใจว่าเส้นทางทั้งหมดได้รับการตั้งค่าอย่างถูกต้อง เส้นทางที่ไม่ถูกต้องอาจนำไปสู่ `FileNotFoundException`-
- ตรวจสอบว่า GroupDocs.Signature ได้รับการติดตั้งและได้รับอนุญาตอย่างถูกต้อง
- ตรวจสอบข้อยกเว้นระหว่างการดำเนินการลงนามเพื่อระบุปัญหาการกำหนดค่า

1. **การจัดการสัญญา**:ทำให้การลงนามสัญญาในองค์กรเป็นระบบอัตโนมัติ ลดระยะเวลาในการดำเนินการด้วยตนเอง

3. **เอกสารทางกฎหมาย**:ลงนามเอกสารทางกฎหมายอย่างปลอดภัย เช่น ข้อตกลงหรือคำให้การโดยไม่ต้องมีเจ้าหน้าที่อยู่ด้วย
