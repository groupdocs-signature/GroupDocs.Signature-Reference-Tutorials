# Учебные пособия по загрузке и сохранению документов с помощью GroupDocs.Signature для .NET

Наши руководства по загрузке и сохранению документов содержат исчерпывающую информацию по программной обработке документов в .NET. Следуйте этим пошаговым инструкциям, чтобы научиться загружать документы с локального диска, из потоков, URL-адресов, облачных хранилищ, таких как Amazon S3 и Azure Blob Storage, а также с FTP. В этих руководствах также рассматриваются различные сценарии сохранения документов, включая сохранение с защитой паролем, различные форматы вывода и специальные параметры сохранения для разных типов документов.
