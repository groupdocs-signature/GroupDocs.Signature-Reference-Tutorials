## Руководство по внедрению

### Обновление текстовых подписей в документах

Обновление текстовых подписей включает поиск существующих подписей и изменение их свойств. Рассмотрим эти шаги:

#### Шаг 1: Подготовьте окружающую среду

Убедитесь, что путь к документу и выходной каталог определены правильно:

```csharp
string filePath = Path.Combine("YOUR_DOCUMENT_DIRECTORY", "SampleSignedMultiDocument.pdf");
string fileName = Path.GetFileName(filePath);
string outputFilePath = Path.Combine("YOUR_OUTPUT_DIRECTORY", "UpdateTextAfterSearch", fileName);
Directory.CreateDirectory(Path.GetDirectoryName(outputFilePath));
File.Copy(filePath, outputFilePath, true);
```

#### Шаг 2: Инициализация и поиск текстовых подписей

Используйте `Signature` класс для поиска текстовых подписей:

```csharp
using (Signature signature = new Signature(outputFilePath))
{
    TextSearchOptions searchOptions = new TextSearchOptions();
    List<TextSignature> foundSignatures = signature.Search<TextSignature>(searchOptions);
```

Этот фрагмент инициализирует объект подписи и ищет текстовые подписи, используя указанные параметры.

#### Шаг 3: Обновите найденные сигнатуры

Пройдитесь по каждой найденной сигнатуре, чтобы обновить ее свойства:

```csharp
foreach (TextSignature temp in foundSignatures)
{
    if (temp.Text == "Text signature")
    {
        // Обновить положение и содержание подписи
        temp.Left += 100;
        temp.Top += 100;
        temp.Text = "Mr. John Smith";
    }
    
    // Отметить как действительную подпись для обновления
    temp.IsSignature = true;
}
```

#### Шаг 4: Примените обновления

Примените все изменения с помощью `Update` метод:

```csharp
UpdateResult updateResult = signature.Update(foundSignatures.ConvertAll(p => (BaseSignature)p));

if (updateResult.Succeeded.Count == foundSignatures.Count)
{
    Console.WriteLine("\nAll signatures were successfully updated!");
}
else
{
    Console.WriteLine($"Successfully updated {updateResult.Succeeded.Count} signatures.");
    Console.WriteLine($"Not updated signatures : {updateResult.Failed.Count}");
}

foreach (BaseSignature temp in updateResult.Succeeded)
{
    Console.WriteLine($"Signature# Id: {temp.SignatureId}, Location: {temp.Left}x{temp.Top}. Size: {temp.Width}x{temp.Height}");
}
```

На этом процесс обновления завершается, и все запланированные сигнатуры изменяются.

### Советы по устранению неполадок

- **Доступ к файлам**: Убедитесь, что у вас есть разрешения на чтение и запись для путей к документам.
- **Поиск по подписи**: Проверьте критерии поиска в `TextSearchOptions` для точного сопоставления желаемых подписей.
- **Ошибки обновления**: Проверьте журналы ошибок, если обновления не применяются, так как некоторые свойства могут быть заблокированы или недействительны.
