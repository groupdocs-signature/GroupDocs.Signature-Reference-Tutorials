## Руководство по внедрению

### Поиск по текстовой подписи

#### Обзор

Эта функция позволяет искать текстовые подписи на всех страницах документа, что упрощает проверку наличия и местоположения подписанного содержимого.

#### Пошаговая реализация

1. **Определить параметры поиска**
   
   Настройте параметры поиска текстовых подписей на всех страницах:
   
   ```csharp
   using GroupDocs.Signature.Options;
   
   TextSearchOptions textOptions = new TextSearchOptions() { AllPages = true };
   ```

2. **Выполнить поиск**
   
   Используйте эти параметры для выполнения поиска по текстовой подписи:
   
   ```csharp
   SearchResult result = signature.Search(textOptions);
   ```

3. **Результаты процесса**
   
   Перебрать найденные сигнатуры и отобразить подробности:
   
   ```csharp
   if (result.Signatures.Count > 0) {
       foreach (var resSignature in result.Signatures) {
           Console.WriteLine($"Text signature found at page {resSignature.PageNumber} with type {resSignature.SignatureType} and Id#: {resSignature.SignatureId}");
       }
   } else {
       Console.WriteLine("No text signatures were found.");
   }
   ```

### Поиск цифровой подписи

#### Обзор

Подобно текстовому поиску, эта функция позволяет находить цифровые подписи по всему документу.

#### Пошаговая реализация

1. **Определить параметры поиска**
   
   Настройте параметры для комплексного поиска:
   
   ```csharp
   using GroupDocs.Signature.Options;
   
   DigitalSearchOptions digitalOptions = new DigitalSearchOptions() { AllPages = true };
   ```

2. **Выполнить поиск**
   
   Выполните поиск с заданными вами параметрами:
   
   ```csharp
   SearchResult result = signature.Search(digitalOptions);
   ```

3. **Результаты процесса**
   
   Вывести подробную информацию о найденных подписях:
   
   ```csharp
   if (result.Signatures.Count > 0) {
       foreach (var resSignature in result.Signatures) {
           Console.WriteLine($"Digital signature found at page {resSignature.PageNumber} with type {resSignature.SignatureType} and Id#: {resSignature.SignatureId}");
       }
   } else {
       Console.WriteLine("No digital signatures were found.");
   }
   ```
