## Руководство по внедрению

Теперь, когда наша настройка готова, давайте реализуем загрузку документов с FTP-сервера с помощью GroupDocs.Signature.

### Подключение и извлечение файлов с FTP

#### Обзор
В этом разделе объясняется, как установить соединение с FTP-сервером и извлекать файлы в виде потоков для обработки в Java.

#### Шаг 1: Настройка FTP-подключения

```java
import org.apache.commons.net.ftp.FTPClient;
import java.io.InputStream;

public class FtpLoader {
    private static InputStream getFileFromFtp(String server, String filePath) throws Exception {
        // Создать экземпляр FTP-клиента
        FTPClient client = new FTPClient();

        // Подключитесь к FTP-серверу
        client.connect(server);

        // Извлечь файл как поток с указанного пути на FTP-сервере
        return client.retrieveFileStream(filePath);
    }
}
```

**Объяснение:**
- **FTP-клиент:** Облегчает операции FTP с использованием Apache Commons Net.
- **RetrearFileStream:** Подключается к FTP-серверу и извлекает файл по адресу `filePath` как входной поток.

#### Шаг 2: Загрузите документ в GroupDocs.Signature

```java
import com.groupdocs.signature.domain.signatures.QrCodeSignature;
import com.groupdocs.signature.options.sign.QrCodeSignOptions;

// Инициализируйте объект Signature с помощью полученного InputStream.
InputStream inputStream = getFileFromFtp("ftp.example.com", "/path/to/document.pdf");
signature.setDocument(inputStream);

// Пример добавления подписи QR-кода в документ
QrCodeSignOptions signOptions = new QrCodeSignOptions("Sample QR Code")
    .setEncodeType(QrCodeTypes.QR)
    .setLeft(100)
    .setTop(100);

signature.sign("signed-document.pdf", signOptions);
```

**Объяснение:**
- **Signature.setDocument:** Устанавливает поток документов для подписания.
- **QrCodeSignOptions:** Настраивает свойства и положение QR-кода в документе.

### Советы по устранению неполадок

- Убедитесь, что ваши учетные данные и пути к FTP-серверу верны.
- Проверьте сетевое подключение к FTP-серверу.
- Обрабатывайте исключения корректно, используя блоки try-catch, чтобы избежать сбоев приложения.
